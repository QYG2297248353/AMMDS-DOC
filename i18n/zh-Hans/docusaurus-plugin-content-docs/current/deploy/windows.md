# Windows

Microsoft Windows，通称Windows操作系统，是微软以图形用户界面为主推出的一系列专有商业软件操作系统。

<!-- truncate -->

## 下载

![下载](./img/deploy/install-windows-00.png)

发行地址：[AMMDS Launcher](https://github.com/QYG2297248353/AMMDS-Launcher/releases)

文件说明：
- `ammds-setup.exe`：AMMDS Launcher的安装程序文件。
- `ammds.zip`: 绿色免安装版本，解压后即可运行。

下载文件：`ammds-setup.exe`

## 安装
双击下载的`ammds-setup.exe`文件，按照提示进行安装。

![安装](./img/deploy/install-windows-01.png)
选择安装语言为简体中文。

![安装](./img/deploy/install-windows-02.png)
选择附加组件，建议安装所有组件。

![安装](./img/deploy/install-windows-03.png)
点击安装

![安装](./img/deploy/install-windows-04.png)
安装进度

![安装](./img/deploy/install-windows-05.png)
安装完成后，点击“完成”。

之后将自动启动AMMDS Launcher。并显示官方文档首页。

![安装](./img/deploy/install-windows-06.png)
如果发现页面为白屏，说明是首次启动，后台正在下载必要的资源文件。请耐心等待，下载完成后，页面将自动刷新。

![安装](./img/deploy/install-windows-07.png)
进入AMMDS Launcher主界面。

## 升级

升级时，直接运行新下载的`ammds-setup.exe`文件，按照提示进行覆盖升级即可。

## 托盘
![安装](./img/deploy/install-windows-08.png)
安装完成后，在系统托盘图标上点击右键。
选择“打开数据目录”，即可查看AMMDS的资源存放目录。
选择“启动”，即可启动AMMDS Launcher。
选择“重启”，即可重启AMMDS Launcher。
选择“停止”，即可停止AMMDS Launcher。
选择“退出”，即可退出AMMDS Launcher。
选择“打开面板”，即可打开AMMDS Launcher的主界面。
选择“打开自动启动”，即可设置AMMDS Launcher在系统启动时自动启动。

## 运行日志

![安装](./img/deploy/install-windows-09.png)
数据目录：
- `data`: 存储AMMDS资源文件
- `db`: 存储AMMDS的数据库文件。
- `logs`: 存储AMMDS的运行日志文件。

![安装](./img/deploy/install-windows-10.png)
资源下载日志，记录了AMMDS Launcher在启动时下载的资源文件。进度达到100%后，说明所有资源文件都已下载完成。
首次启动时会下载多个资源文件，可能需要一定的时间。请耐心等待。
