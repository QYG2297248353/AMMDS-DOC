# NexusPHP

NexusPHP 由浙江大学的 Nexus 团队发起并开发完成，是一款专注于提供完整、有序、重视用户信誉和知识的资源分享社区解决方案的 PT（Private Tracker，即私有的 Tracker 服务器，是 BT 下载的一种）开源软件。

:::info 私有站点说明
该站点需要通过邀请注册，具有一定门槛。
:::

## 插件配置

![插件配置](/img/plugin/nexusphp-01.png)

### 基本配置

- **启用状态**：控制是否启用 NexusPHP 插件。
- **检索功能**：是否启用 NexusPHP 的检索功能，默认值为关闭状态。
- **同步用户信息**：是否启用 NexusPHP 的用户信息同步功能，默认值为关闭状态。用户信息用于站点榜展示。
- **告警通知**：是否启用 NexusPHP 的告警通知功能，默认值为关闭状态。

![告警通知](/img/plugin/nexusphp-02.png)

## NexusPHP 站点模块

点击右上角「添加站点」按钮，进行站点添加。

:::warning 版本要求
NexusPHP 站点版本必须在 1.9.0 以上。

**查看版本方法**：在 NexusPHP 站点页面底部 `Powered by NexusPHP` 处，通过点击 NexusPHP 查看站点版本。
:::

![添加站点](/img/plugin/nexusphp-03.png)

### 站点配置项

- **是否启用**：是否启用该站点，默认值为关闭状态。
- **站点名称**：站点名称，用于在插件中显示。除手动输入外，可通过下拉选择常见站点名称，系统会自动填充站点地址。
- **站点地址**：站点地址，用于访问站点，默认值为空。

![站点令牌](/img/plugin/nexusphp-04.png)

- **站点令牌**：在「控制面板 >> 設定首页 >> 访问令牌」中通过创建按钮进行创建。

:::info 权限要求
**默认权限**：获取种子列表、查看种子详情、发布种子、查看用户基本信息。

**最低权限要求**：获取种子列表、查看种子详情、查看用户基本信息。

**权限不足**：缺少权限时，系统会提示 `权限不足`，无法正常添加此站点。
:::

- **站点 Logo**：站点 Logo，用于在插件中显示，默认值为空。
