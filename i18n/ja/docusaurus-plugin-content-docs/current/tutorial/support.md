# サポート

🥳 一緒に成長しましょう！

AMMDS サポートページへようこそ！ご関心とご支援をありがとうございます。皆様からのサポートは AMMDS の改善につながり、より良い体験を提供することに役立ちます。

## どうやってサポートできますか？

### 1. メンバーになる

オンラインストアで AMMDS 関連の製品やサービスをお買い求めください。良い製品を手に入れながら、私たちの活動を支援できます。

### 2. 友達にシェアする

AMMDS が気に入ったら、家族や友人に紹介してください！シェアが最大の励みになり、より多くの人にこのツールを使ってもらえます。

### 3. データを提供する

AMMDS のデータは共有されており、誰でも表示、修正、改善提案ができます。問題があれば GitHub で報告してください。

### 4. 提案をする

あなたの意見は私たちにとって重要です。新機能のリクエスト、バグ報告、その他のアイデアがあれば、いつでもお知らせください。公式サイトから連絡するか、GitHub で直接 Issue を送ってください。

### 5. スポンサーになる

私たちの活動を気に入り、継続的な発展を支援したい場合は、プロジェクトへの寄付をご検討ください。あなたの支援はサーバー費用やドメイン費用などに役立ちます。

## 私たちの約束

私たちは高品質な製品とサービスを提供するために努力し続けます。チームは皆さんのニーズに応え、より良い体験を届けるために改善を続けます。

AMMDS は映画やテレビ番組のファンのために作られたデータ管理プラットフォームです。あなたの個人コレクションを簡単に管理・楽しめます。メタデータの自動取得、ローカルファイルのスマートマッチング、クロスプラットフォーム対応まで、AMMDS は最高の体験を提供することを目指しています。
