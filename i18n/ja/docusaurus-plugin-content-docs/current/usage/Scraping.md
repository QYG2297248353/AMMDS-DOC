# スクレイピングフローの詳細説明

このドキュメントでは、AMMDS のスクレイピングフロー全体をわかりやすく説明します。メディアディレクトリのマウントから、ファイルのスキャン、動画番号の識別、メタデータのスクレイピング、ファイルの整理まで、すべてのステップをカバーします。
