# スキャン設定

スキャン設定は、システムに次のことを伝えます：ファイルをスキャンするときに、どのファイルを処理し、どのファイルをスキップし、どのようにメディアを識別するか。

## スキャン設定へのアクセス

以下のパスからスキャンルール設定ページにアクセスできます：

**タスク管理 >> スキャン規則設定**

:::info 操作のヒント
設定を変更したら忘れずに **保存** をクリックしてください。保存しないと反映されません。
:::

## 設定項目の説明

### 形式制限

**干啥的（なにをするもの）**：どのファイル形式をメディアファイルとみなすかをシステムに伝えます。システムはこれらの形式のみを認識します。

**デフォルト設定**：システムには `.mp4`, `.mkv`, `.avi`, `.mov` などの一般的な形式があらかじめ登録されています。

**自分で追加**：新しい形式を追加するには、拡張子（例：`.strm`）を入力して Enter キーで確定します。

### 無視ディレクトリ

**干啥的**：スキャンしたくないディレクトリ（サンプルや一時ファイルのフォルダなど）があれば、ここで設定してスキップできます。

**設定方法**：ディレクトリの相対パス（例：`sample`, `tmp`）を入力します。大文字小文字は区別しません。

**どんなときに使うか**：ダウンロードフォルダの中にある `sample` ディレクトリにはサンプル動画だけが入っていて、それをスキャンしたくない場合など。

### 無視ファイル

**干啥的**：ファイル名に特定の文字列（例：`sample`）が含まれているファイルをスキャンしたくない場合に設定します。

**設定方法**：ファイル名に含まれる文字列（例：`sample`, `hh800.com@`）を入力します。大文字小文字は区別しません。

**どんなときに使うか**：ファイル名に広告マークが付いているものを無視したい場合など。

### 分割マーク

**干啥的**：動画が複数のパート（CD1、CD2 など）に分割されている場合、システムがそれらをひとつの作品として認識し、個別のファイルとして扱わないようにするための設定です。

**設定方法**：分割マーク（例：`part`, `cd`）を入力します。大文字小文字は区別しません。

**どんなときに使うか**：映画が前後編に分かれていたり、複数のファイルに分割されている場合。

### 字幕形式

**干啥的**：システムが認識できる字幕ファイルの形式を設定します。

**設定方法**：字幕ファイルの拡張子（例：`.srt`, `.ass`, `.idx`）を入力して Enter キーで確定します。

### 最小動画サイズ

**干啥的**：サイズの下限を設定します。これより小さい動画ファイルは無視されます（小さすぎるものはまともな映画ではない可能性が高いため）。

**単位**：MB

**デフォルト値**：100MB

**特殊なケース**：`.strm` のようなファイル（リンクのみで実体がない）を使う場合は、最小値は **0MB** に設定することをおすすめします。設定しないとスキャンできません。

### 中国語字幕の識別

**干啥的**：動画ファイル名に特定のキーワードが含まれている場合、システムが自動的に「中国語字幕」のタグを付けます。

**例**：`-C`, `.chs` など。大文字小文字は区別しません。

### モザイク解除の識別

**干啥的**：動画ファイル名に特定のキーワードが含まれている場合、システムが自動的に「モザイク解除」のタグを付けます。

**例**：`-u`, `.uc` など。大文字小文字は区別しません。

### ローカル nfo の読み取り

**干啥的**：この機能を有効にすると、システムは動画ファイルと同じ場所にある同名の `.nfo` ファイルを自動的に読み込み、その情報を解析します。

**有啥用（どういう効果）**：解析された情報は自動的にデータベースに保存されるので、作品情報を手動で入力する手間が省けます。

## 設定例

典型的なスキャン設定の例です：

| 設定項目 | 設定値 |
|--------|--------|
| 形式制限 | `.mp4`, `.mkv`, `.avi`, `.mov`, `.strm` |
| 無視ディレクトリ | `sample`, `tmp`, `Extras` |
| 無視ファイル | `sample`, `trailer`, `preview` |
| 分割マーク | `part`, `cd`, `disc` |
| 字幕形式 | `.srt`, `.ass`, `.idx`, `.sub` |
| 最小動画サイズ | 100MB |
| 中国語字幕の識別 | `-C`, `.chs`, `.zh` |
| モザイク解除の識別 | `-u`, `.uc`, `uncensored` |
| ローカル nfo の読み取り | 有効 |

## よくある質問

### Q: 一部の動画ファイルがスキャンされないのはなぜ？

**A**：以下の原因が考えられます：
- ファイル形式が形式制限リストに含まれていない
- ファイルのあるディレクトリが「無視ディレクトリ」に設定されている
- ファイル名に無視対象の文字列が含まれている
- ファイルサイズが小さすぎて最小動画サイズに達していない

### Q: カスタムファイル形式を正しく追加するには？

**A**：追加する際の注意点：
- 拡張子にはドットを含める（例：`.strm` であって `strm` ではない）
- 各形式は個別に追加し、追加後に Enter キーで確定する
- 追加する形式がシステムで処理可能であることを確認する

### Q: 分割マークの役割は？

**A**：分割マークは、同じ作品に属する複数の分割ファイルをシステムが正しく認識し、ひとつの作品として扱うために使います。複数の無関係なファイルとして扱われるのを防ぎます。
