# スキャンルール設定

スキャンルール設定は、メディアファイルのスキャンルールを設定するために使用され、ファイル形式のフィルタリング、ディレクトリの除外などの設定を含み、必要なメディアファイルを正確にスキャンするのに役立ちます。

<!-- truncate -->

## スキャンルール設定へのアクセス

以下のパスからスキャンルール設定ページにアクセスすることができます：

**タスク管理 >> スキャンルール**

## 機能説明

### スキャンルール

**機能説明**：スキャンルールを作成することで、スキャンするファイルと除外するファイルを指定することができ、スキャンの効率と精度を向上させることができます。

**適用シナリオ**：
- 特定のタイプのメディアファイルのみをスキャンしたい場合
- 特定のディレクトリやファイルを除外したい場合
- 特定のスキャン条件を設定したい場合

## 設定方法

### スキャンルールの作成

1. **スキャンルール**ページで、**作成**ボタンをクリックします
2. ルール名と設定パラメータを入力します
3. ルールを保存します

### ルールパラメータ

#### ファイル形式

**機能説明**：スキャンするファイル形式を設定します。

**設定推奨**：スキャンしたいメディアファイル形式（MP4、MKV、AVI など）を選択してください。

#### 除外ディレクトリ

**機能説明**：スキャンする必要のないディレクトリを設定します。

**設定推奨**：メディアファイルを含まないディレクトリ（システムディレクトリ、一時ディレクトリなど）を追加してください。

#### 除外ファイル

**機能説明**：スキャンする必要のないファイルを設定します。

**設定推奨**：メディアファイルではないファイル（テキストファイル、画像ファイルなど）を追加してください。

#### 最小ファイルサイズ

**機能説明**：スキャンするファイルの最小サイズを設定します。

**設定推奨**：メディアファイルではない小さなファイルを除外するために適切な値を設定してください。

## 設定例

以下は、スキャンルールのいくつかの例です：

| ルール名 | ファイル形式 | 除外ディレクトリ | 除外ファイル | 最小ファイルサイズ |
|---------|-------------|-----------------|-------------|-------------------|
| 動画ファイル | MP4, MKV, AVI | temp, system | *.txt, *.jpg | 100MB |
| 高品質動画 | MP4, MKV | downloads/incomplete | *.sample.* | 500MB |
| すべてのメディア | MP4, MKV, AVI, MOV, WMV | - | - | 50MB |

## よくある質問

### Q: スキャンルールの目的は何ですか？

**A**：スキャンルールは、スキャンプロセス中にメディアファイルをフィルタリングして識別するために使用され、システムが処理する必要のあるメディアファイルを正確に見つけるのを助け、スキャン効率を向上させ、不要な処理を減らします。

### Q: 効果的なスキャンルールを設定するにはどうすればよいですか？

**A**：以下の方法で効果的なスキャンルールを設定することができます：
1. スキャンしたいファイル形式を明確に定義します
2. 不要なディレクトリをスキャンしないように合理的に除外ディレクトリを設定します
3. 非メディアファイルを除外するために適切な最小ファイルサイズを設定します
4. 実際の使用状況に基づいて、スキャンルールを定期的に更新および最適化します

### Q: 複数のスキャンルールを作成できますか？

**A**：はい、異なるシナリオに対して複数のスキャンルールを作成することができ、スキャンタスクを作成する際に適切なスキャンルールを選択することができます。

### Q: スキャンルールを選択しない場合はどうなりますか？

**A**：スキャンルールを選択しない場合は、システムはデフォルトのスキャンルールを使用します。デフォルトのスキャンルールは通常、一般的なメディアファイル形式をスキャンし、どのディレクトリやファイルも除外しません。

### Q: スキャンルールが有効かどうかをテストするにはどうすればよいですか？

**A**：以下の方法でテストすることができます：
1. スキャンルールを設定します
2. スキャンタスクを作成し、スキャンルールを選択します
3. スキャンタスクを実行します
4. スキャン結果が期待どおりかどうかを確認します

## ベストプラクティス

- **明確なルール命名**：後続の管理のために説明的なルール名を使用します
- **合理的なファイル形式設定**：実際に必要なメディアファイル形式のみを選択します
- **効果的なディレクトリ除外**：メディアファイルを確実に含まないディレクトリを追加してスキャン速度を向上させます
- **適切なファイルサイズ設定**：非メディアファイルをフィルタリングするために合理的な最小ファイルサイズを設定します
- **定期的なルールメンテナンス**：実際の使用状況に基づいて、スキャンルールを定期的に確認および更新します
