# スクレイピング設定

スクレイピング設定は、システムに次のことを伝えます：ネットから作品情報（表紙、あらすじ、俳優など）を探すときにどう探すか、整理後のファイルをどんな形式で保存するか。

> **「スクレイピング」って何？** スクレイピングとは、システムが自動的にネット上で作品の情報（タイトル、表紙画像、キャスト、あらすじなど、まとめて「メタデータ」と呼びます）を検索し、ダウンロードしてあなたの動画ファイルと紐づけることです。

## スクレイピング設定へのアクセス

以下のパスからスクレイピングルール設定ページにアクセスできます：

**タスク管理 >> スクレイピング規則設定**

:::info 操作のヒント
設定を変更したら忘れずに **保存** をクリックしてください。保存しないと反映されません。
:::

## 基本設定

### 文字フィルタリング

**干啥的（なにをするもの）**：スクレイピング前にファイル名の不要な文字を除去し、識別精度を向上させます。

**有啥用（どういう効果）**：ファイル名のノイズを取り除くことで、システムがより正確に作品番号を認識し、より正確な作品情報を見つけられます。

### 画像ダウンロード失敗を許可

**干啥的**：画像のダウンロードに失敗しても、スクレイピングタスクを続行させます。

**有啥用**：このオプションをオンにすると、一部の画像がダウンロードできなくてもスクレイピング結果に影響せず、全体的な成功率が向上します。

### 整理ディレクトリ形式

**干啥的**：スクレイピング完了後にフォルダをどのように構成するかを設定します。

**怎么配（設定方法）**：システムが提供する変数を組み合わせて、好きなディレクトリ形式を作ります。

**例**：`${actor}/${number}` は「俳優名/作品番号」という 2 階層のディレクトリ構造になります。

### 整理ファイル名形式

**干啥的**：スクレイピング完了後に動画ファイルをどのように命名するかを設定します。

**怎么配**：システムが提供する変数を組み合わせて、好きなファイル名形式を作ります。

**例**：`${title_safe}.${ext}` は安全な作品タイトルをファイル名に使い、元の拡張子を保持します。

### 自動スクレイピング

**干啥的**：スキャンタスク完了後に、自動的にスクレイピングを開始するかどうかを設定します。

**デフォルトの動作**：デフォルトでは手動でファイルを選んでスクレイピングする必要があります。

**有効化の効果**：オンにすると、システムが自動的にスクレイピング可能な動画ファイルを見つけて処理を開始します。

### 自動スクレイピング間隔

**干啥的**：自動スクレイピングをどのくらいの間隔で実行するかを設定します。

**単位**：分

**デフォルト値**：3分（自動スクレイピングが有効な場合）

### 高解像度カバーのダウンロード

**干啥的**：作品の高解像度カバー画像をダウンロードするかどうかを設定します。

**ダウンロード元**：オンにすると、まず Fanza DMM から高解像度カバーを取得しようとします。見つからなければ他のソースを探します。

### 外部字幕の検索

**干啥的**：作品の外部字幕（ソフト字幕）を検索してダウンロードするかどうかを設定します。

**保存場所**：ダウンロードされた字幕ファイルは最終的なスクレイピングディレクトリに保存されます。

### 番号の統合

**干啥的**：スクレイピング後に生成される nfo ファイルのタイトルなどのフィールドに、番号のプレフィックスを追加するかどうかを設定します。

### 自動タグ

**干啥的**：スクレイピング後に生成される nfo ファイルの tag タグに追加情報を加えるかどうかを設定します。

**例**：解像度、番号、中文字幕、モザイク解除などのタグ。

### 自動カテゴリタグ

**干啥的**：スクレイピング後に生成される nfo ファイルの genre タグに追加情報を加えるかどうかを設定します。

**例**：解像度、番号、中文字幕、モザイク解除などのタグ。

### ローカルデータ優先

**干啥的**：スクレイピング時にローカルに既存のデータを優先して使用するかどうかを設定します。

**有啥用**：ローカルデータが要件を満たしている場合は、ネットからダウンロードせずにローカルデータを直接使うため、スクレイピングが速くなります。

### スクレイパー

**干啥的**：どの情報ソースを使って作品情報を検索するかを選択します。

**選択肢**：
- **ローカル**：自分で手動作成した作品情報を使用
- **Metatube** (プラグイン)
- **ThePornDB** (プラグイン)
- **Stash** (プラグイン)
- **Fanza DMM** (プラグイン)

### スクリーンショットのスクレイピング

**干啥的**：作品のスクリーンショット（シーン画像）をダウンロードするかどうかを設定します。

**保存場所**：ダウンロードされたスクリーンショットは指定したディレクトリに保存されます。

### スクリーンショットディレクトリ

**干啥的**：スクリーンショットファイルの保存先フォルダを設定します。

**デフォルト値**：動画ファイルと同じディレクトリの `extrafanart` フォルダ。

### マルチスレッド実行

**干啥的**：複数のスレッドを同時に使ってスクレイピングするかどうかを設定します（複数ファイルを同時処理して高速化）。

**メリット・デメリット**：
- **メリット**：スクレイピングが速くなる
- **デメリット**：システムリソースをより消費する

**推奨**：お使いの PC スペックに応じて設定してください。CPU が 4 コアなら 4 スレッドを推奨。

### 最大スレッド数

**干啥的**：スクレイピング時に同時に使用する最大スレッド数を設定します。

**デフォルト値**：5 スレッド

**推奨**：PC スペックに応じて設定し、高く設定しすぎてシステムが重くならないように注意してください。

## ディレクトリとファイル名の整理

設定には **整理ディレクトリ形式** と **整理ファイル名形式** の 2 つのオプションがあります。システムが提供する変数を組み合わせて、ディレクトリ構成やファイル名をカスタマイズできます。

### セーフ変数

**干啥的**：変数名の後ろに `_safe` サフィックスを付けると、ファイル名に特殊記号が含まれることによる問題を回避できます。

**例**：`${actor_safe}` は俳優名に特殊文字が含まれている場合、安全な文字に置き換えます。

**有啥用**：作品タイトルの特殊記号がディレクトリ構造を壊すのを防ぎます。

### 整理例

動画ファイルは次のように整理されます：`${actor}/${number}/${title_safe}.${ext}`

**例**：
- 俳優名："John Doe"
- 作品番号："ABC-123"
- 作品タイトル："Sample Title"
- ファイル拡張子："mkv"

**最終パス**：`John Doe/ABC-123/Sample Title.mkv`

### 整理ディレクトリ変数

**デフォルト設定**：`${actor}/${number}`

**注意**：`/` は区切り文字として扱われ、異なるディレクトリ階層を作成できます。

| 変数名 | セーフ変数名 | 説明 | 例 |
|--------|------------|------|------|
| `${actor}` | `${actor_safe}` | 俳優名 | "John Smith" |
| `${number}` | `${number_safe}` | 作品番号 | "ABC-123" |
| `${title}` | `${title_safe}` | 作品タイトル | "Sample Title" |
| `${date}` | `${date_safe}` | 発売日 | "2024/01/01" |
| `${year}` | `${year_safe}` | 発売年 | "2024" |
| `${month}` | `${month_safe}` | 発売月 | "01" |
| `${day}` | `${day_safe}` | 発売日 | "01" |
| `${mosaic}` | `${mosaic_safe}` | モザイクタイプ | "無修正/修正" |
| `${resolution}` | `${resolution_safe}` | 動画解像度 | "1080p" |
| `${director}` | `${director_safe}` | 監督名 | "Director Name" |
| `${studio}` | `${studio_safe}` | 制作会社名 | "Studio Name" |

**デフォルト値**：すべての変数にはデフォルト値があり、指定がない場合はデフォルト値が使われます。

## よくある質問

### Q: スクレイピングに失敗するのはなぜ？

**A**：以下の原因が考えられます：
- 番号が正しく認識されていない
- ネットに接続できない
- スクレイパーが一致する情報を見つけられない
- 設定が間違っている

### Q: スクレイピングの成功率を上げるには？

**A**：以下の方法を試してください：
- ファイル名の番号がはっきり識別できるようにする
- 文字フィルタリング機能を有効にする
- 複数のスクレイパーを設定する
- ネットワーク接続が正常であることを確認する

### Q: 自動スクレイピングと手動スクレイピングの違いは？

**A**：
- **自動スクレイピング**：スキャン完了後に自動開始。手間いらずで一括処理に最適
- **手動スクレイピング**：自分でファイルを選んでスクレイピング。結果を細かく調整したい場合に最適

### Q: ダウンロードした字幕が動画と合わないのはなぜ？

**A**：以下の原因が考えられます：
- 字幕検索時の番号認識が正確でない
- ネットワークの問題で字幕がダウンロードできなかった
- 字幕ファイルの名前が動画ファイルと一致していない

### Q: ディレクトリ構造をカスタマイズするには？

**A**：システム提供の変数を組み合わせます。例えば：
- `${studio}/${actor}/${number}`：「制作会社 → 俳優 → 番号」で整理
- `${year}/${month}/${title_safe}`：「年 → 月 → タイトル」で整理
