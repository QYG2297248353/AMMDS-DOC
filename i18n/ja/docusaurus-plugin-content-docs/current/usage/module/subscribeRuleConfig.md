# サブスクリプションルール設定

サブスクリプションルール設定は、システムに次のことを伝えます：どんなリソースが見たいか、どこから探すか、どこにダウンロードするか。システムがあなたの好みに合わせて自動的に動きます。

<!-- truncate -->

## サブスクリプションルール設定へのアクセス

以下のパスからアクセスできます：

**タスク管理 >> サブスクリプションルール**

## 設定インターフェース

![サブスクリプションルール設定](/img/usage/module/subscribe-config-01.png)

## 設定項目の説明

### 中文字幕優先

**干啥的（なにをするもの）**：オンにすると、システムは中文字幕または中国語吹き替え版を優先的に検索します。

**啥时候用（いつ使うか）**：中国語版を視聴したい場合は、これをオンにすると中国語版を入手できる確率が上がります。

### 検索順序

**干啥的**：検索ソースの優先順位を設定します。システムは指定した順序に従って、各検索ソースから順番にリソースを探します。

**建议（推奨）**：リソースの品質が高く、更新が早い検索ソースを上位に設定すると、効率的に高品質なリソースを取得できます。

### ダウンロードディレクトリ

**干啥的**：ダウンローダーがファイルを保存するフォルダを指定します。

**重要な注意事項**：

:::warning 注意事項
- このパスは AMMDS システム自体とは直接関係ありません。使用しているダウンローダーの設定に応じて設定する必要があります
- 指定したディレクトリが存在し、書き込み権限があることを確認してください
- 相対パスによるパス解釈エラーを避けるため、絶対パス（例：`D:\Downloads\Movies`）の使用を推奨します
:::

## 設定例

典型的なサブスクリプションルール設定の例です：

1. **中文字幕優先**：有効
2. **検索順序**：検索ソース A → 検索ソース B → 検索ソース C
3. **ダウンロードディレクトリ**：`D:\Downloads\Movies`

## よくある質問

### Q: 中文字幕優先を設定したのに、中文字幕なしのバージョンがダウンロードされるのはなぜ？

**A**：そのリソースにまだ中国語字幕版がリリースされていない可能性があります。システムは中国語字幕版がない場合、他のバージョンを先にダウンロードして、タイムリーにリソースを入手できるようにします。

### Q: ダウンロードディレクトリの設定を間違えるとどうなる？

**A**：以下の問題が発生する可能性があります：
- ダウンローダーがファイルを正常に保存できない
- システムがダウンロード済みファイルを見つけられない
- ストレージ不足でダウンロードに失敗する

ダウンロードディレクトリのパスは正しく設定してください。
