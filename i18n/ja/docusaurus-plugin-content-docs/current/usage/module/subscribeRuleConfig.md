# サブスクリプションルール設定

サブスクリプションルール設定は、タスク実行前に、動画コンテンツのサブスクリプションの好みとダウンロードパラメータを設定するためのもので、システムがお客様の要件に従って動画リソースを自動的に取得・管理できるようにします。

<!-- truncate -->

## サブスクリプションルール設定へのアクセス

サブスクリプションルール設定ページには、以下のパスからアクセスできます：

**タスク管理 >> サブスクリプションルール**

## 設定項目の説明

### 中国語字幕優先

**機能説明**：このオプションを有効にすると、システムは中国語字幕または中国語吹き替えの動画バージョンを優先的に検索・ダウンロードします。

**適用シナリオ**：中国語コンテンツをより好む場合は、このオプションを有効にすることで、中国語バージョンを取得する確率が向上します。

### 検索順序

**機能説明**：検索ソースの優先順位を設定します。システムは指定された順序に従って、各検索ソースから順次動画リソースを取得します。

**設定の推奨**：リソースの質が高く、更新速度が速いと思われる検索ソースを上位に配置することで、高品質なリソースを効率的に取得できるようになります。

### ダウンロードディレクトリ

**機能説明**：ダウンローダーのダウンロードディレクトリのパスを指定します。

**重要なヒント**：

:::warning 注意事項
- このパスは AMMDS システム自体とは直接関係がありません。使用しているダウンローダーの設定に応じて設定する必要があります
- 指定されたディレクトリが存在し、書き込み権限があることを確認してください
- 相対パスによるパス解釈エラーを避けるため、絶対パスの使用を推奨します
:::

## 設定例

以下は、典型的なサブスクリプションルール設定の例です：

1. **中国語字幕優先**：有効
2. **検索順序**：検索ソース A → 検索ソース B → 検索ソース C
3. **ダウンロードディレクトリ**：`D:\Downloads\Movies`

## よくある質問

### Q: 中国語字幕優先を設定したのに、非中国語字幕のバージョンがダウンロードされるのはなぜですか？

**A**：現在の動画リソースに中国語字幕のバージョンがまだリリースされていない可能性があります。システムは、中国語字幕のバージョンがない場合には、他のバージョンをダウンロードして、お客様がリソースをタイムリーに取得できるようにします。

### Q: ダウンロードディレクトリの設定エラーはどのような問題を引き起こしますか？

**A**：ダウンロードディレクトリの設定が間違っていると、以下の問題が発生する可能性があります：
- ダウンローダーがファイルを正常に保存できない
- システムがダウンロード完了したファイルを見つけられない
- ストレージ容量不足によるダウンロードの失敗

正しいダウンロードディレクトリのパスを設定するようにしてください。
