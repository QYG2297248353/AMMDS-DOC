# 高德地图

高德地图プラグインは、AMMDS プラットフォームのホームページダッシュボードに天気情報を表示します。高德地图 Web サービス API を呼び出してリアルタイムの天気データを取得します。

## プラグイン設定

![高德地图プラグイン設定](/img/plugin/amap-01.png)

| 設定項目 | 説明 | デフォルト値 | 必須 |
|--------|------|--------|----------|
| 有効化 | プラグインのオン/オフを制御 | オフ | いいえ |
