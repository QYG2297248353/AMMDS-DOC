# R18Dev

R18Dev は R18.com 閉鎖後のデータ保存プロジェクトであり、JAV 動画のメタデータ情報を保持・提供することを目的としています。R18Dev プラグインを通じて、AMMDS は動画のタイトル、出演者、発売日、メーカー、カテゴリなどのメタデータを自動取得し、メディアライブラリ情報を迅速に構築できます。

## プラグイン設定

![基本設定](/img/plugin/r18dev-01.png)

### 基本設定

| パラメータ | 説明 | デフォルト値 |
|------|------|--------|
| 有効化状態 | R18Dev プラグインの有効/無効を制御 | オフ |
| サービスアドレス | R18Dev サービスのアクセスアドレス | `https://r18.dev` |
| 動画検索 | 動画メタデータ検索機能の有効/無効を制御 | オフ |
| オンライン検索 | 有効にするとネットワーク経由での検索を優先 | オフ |
