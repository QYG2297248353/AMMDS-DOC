# Bitmagnet

Bitmagnet はセルフホスト型の BitTorrent DHT クローラー、コンテンツ分類子、トレント検索エンジンで、Go 言語で開発されています。DHT ネットワークをクロールして自動的にトレントを発見し、コンテンツの分類とメタデータの拡張を行い、Web UI、GraphQL API、Torznab 互換インターフェースを提供します。

公式サイト：[https://bitmagnet.io/](https://bitmagnet.io/)

## プラグイン設定

![プラグイン設定](/img/plugin/bitmagnet-01.png)

### 基本設定

- **有効化状態**：プラグインの有効/無効を制御します。
- **サービスアドレス**：Bitmagnet サービスのアクセスアドレス。このプラグインを使用するには、Bitmagnet サービスを自分で構築する必要があります。
