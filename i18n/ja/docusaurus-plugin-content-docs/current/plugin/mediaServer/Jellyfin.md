# Jellyfin

Jellyfin は、デジタルメディアファイルの整理、管理、共有のために設計されたオープンソースのマルチメディア管理アプリケーションスイートです。このプラグインを通じて、AMMDS は Jellyfin メディアサーバーと深度な統合を確立し、メタデータの同期、俳優のプロフィール写真の自動補完などの高度な機能を実現します。

## プラグイン設定

![プラグイン設定](/img/plugin/jellyfin-01.png)

### 基本設定

| パラメータ | 説明 | デフォルト値 |
|------|------|--------|
| 有効状態 | Jellyfin プラグインの有効化を制御します | 無効 |
| サービスアドレス | Jellyfin サーバーのネットワークエンドポイント | `http://127.0.0.1:8096` |
| API キー | Jellyfin サーバーの認証トークン | - |
| ユーザー名 | (オプション) Jellyfin サーバーのログインユーザー名 | - |
| パスワード | (オプション) Jellyfin サーバーのログインパスワード | - |
| ユーザーのバインド | バインドする Jellyfin ユーザーアカウントを選択します | - |
| 自動補完 | メディアライブラリの俳優のプロフィール写真の欠落を自動的に補完するかどうかを制御します | 無効 |

![プラグイン設定](/img/plugin/jellyfin-02.png)

### 設定の説明

- **サービスアドレス**：サーバーの実際のネットワークアドレスを使用してください。`localhost` または `127.0.0.1` の使用は避けてください。そうしないと、コンテナ化されたデプロイメントで接続に失敗する可能性があります。
- **API キー**：Jellyfin 管理インターフェースの「システム」→「詳細」→「API キー」で生成できます。
- **ユーザーのバインド**：正しいサービスアドレスと API キーを入力して保存した後にのみ、ドロップダウンリストからユーザーを選択できます。
- **自動補完**：有効にすると、システムはメディアライブラリの俳優の欠落しているプロフィール写真を自動的に補完します。ただし、AMMDS に対応する俳優のプロフィール写真が追加されている必要があります。

### 接続の検証

- **ネットワーク接続性**：AMMDS サーバーが Jellyfin サービスアドレスにアクセスできることを確認してください。
- **認証情報**：API キーには必要な操作を実行するのに十分な権限が必要です。
- **サービスの状態**：Jellyfin サーバーが実行中で、API リクエストに正常に応答できることを確認してください。

## 機能特性

- **メタデータの同期**：Jellyfin メディアライブラリとのメタデータの双方向同期をサポートします。
- **ユーザーのバインド**：特定の Jellyfin ユーザーをバインドして、パーソナライズされた設定を実現できます。
- **プロフィール写真の自動補完**：俳優の欠落しているプロフィール写真の情報をインテリジェントに識別して補完します。
- **状態の監視**：Jellyfin サーバーの実行状態をリアルタイムで監視します。

## 注意事項

:::warning サービスアドレスの設定
特にコンテナ化されたデプロイメント環境では、`localhost` または `127.0.0.1` をサービスアドレスとして使用しないでください。そうしないと、接続に失敗する可能性があります。
:::

:::info 自動補完の前提条件
自動補完機能を有効にする前に、AMMDS に対応する俳優のプロフィール写真データが追加されていることを確認してください。そうしないと、補完機能が正常に動作しない可能性があります。
:::

:::tip API キーの権限
システムのセキュリティを確保するために、AMMDS 用に専用の API キーを作成し、適切な権限範囲を付与することを推奨します。
:::
