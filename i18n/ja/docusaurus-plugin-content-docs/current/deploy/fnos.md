# 飛牛 FnOS デプロイガイド

飛牛 FnOS は、NAS（家庭用ストレージサーバー）向けに設計されたシステムで、コアなユーザーによって開発されています。AI ローカルモデルを内蔵し、家庭や企業でのデータ管理を便利にします。ここでは飛牛 FnOS に AMMDS をデプロイする方法を紹介します。

<!-- truncate -->

## デプロイ方法

AMMDS は飛牛 FnOS で 2 つのインストール方法を提供しています：

- **FPK オフラインアプリインストール**：飛牛 FnOS のアプリセンターから手動インストール。ほとんどのユーザーに最適
- **Docker インストール**：Docker コンテナでデプロイ。コンテナに詳しいユーザー向け

## FPK オフラインアプリインストール

### ダウンロード

![AMMDS FPK 下载页面](/img/deploy/install-fnos-fpk-00.png)

#### 公式リリースアドレス

[AMMDS Offline FnNas GitHub Releases](https://github.com/QYG2297248353/AMMDS-Offline-FnNas/releases)

#### 利用可能なインストールパッケージ

| ファイル名                  | タイプ | 説明                 |
| ----------------------- | ---- | -------------------- |
| `AMMDS-Offline-FnNas.fpk` | FPK  | AMMDS オフラインインストールパッケージ。飛牛 FnOS システム用 |

#### ダウンロード方法

![AMMDS FPK 下载选项](/img/deploy/install-fnos-fpk-01.png)

インストールパッケージをローカル PC にダウンロードすることも、飛牛デバイス上で直接ダウンロードすることもできます。

### インストール手順

![AMMDS FPK 安装界面](/img/deploy/install-fnos-fpk-02.png)

1. **インストールを開始**

   アプリセンターを開き、左下の**手動インストール**をクリック。ダウンロードした FPK ファイルを選んで**インストール**をクリックします。

![AMMDS FPK 安装提示](/img/deploy/install-fnos-fpk-03.png)

2. **インストールを確認**

   インストール確認画面で**同意**をクリックしてインストールを進めます。

![AMMDS FPK 许可说明](/img/deploy/install-fnos-fpk-04.png)

3. **ライセンスを確認**

   インストールライセンスの内容をよく読んで、条項を理解してください。

![AMMDS FPK 选择安装位置](/img/deploy/install-fnos-fpk-05.png)

4. **インストール場所を選択**

   空き容量が十分にあるディスクを選んでインストールします。

![AMMDS FPK 设置管理员密码](/img/deploy/install-fnos-fpk-06.png)

5. **管理者パスワードを設定**

   AMMDS 管理画面のログインパスワードを設定します。強力なパスワードを推奨します。

![AMMDS FPK 设置访问端口](/img/deploy/install-fnos-fpk-07.png)

6. **アクセスポートを設定**

   AMMDS のアクセスポートを設定します。デフォルトは 9523 です。
   サービス利用規約を読み、**同意**をクリックします。

![AMMDS FPK 开始安装](/img/deploy/install-fnos-fpk-08.png)

7. **インストールを開始**

   最後の確認画面で**確定**をクリックしてインストールを開始します。

![AMMDS FPK 安装进度](/img/deploy/install-fnos-fpk-09.png)

8. **インストール完了を待つ**

   インストール中はプログレスバーが表示されます。しばらくお待ちください。エラーが出た場合は指示に従うか、旧バージョンの FPK を試してみてください。

![AMMDS FPK 安装完成](/img/deploy/install-fnos-fpk-10.png)

9. **インストール成功を確認**

   インストールが完了すると、インストール済みアプリ一覧に AMMDS が表示されます。

### インストールディレクトリ

![AMMDS 安装目录](/img/deploy/install-fnos-fpk-11.png)

ファイル管理 → アプリファイルで、AMMDS のインストールディレクトリを確認できます。

## Docker インストール

### イメージのダウンロード

![AMMDS Docker 镜像搜索](/img/deploy/install-fnos-docker-00.png)

1. **イメージを検索**

   Docker アプリを開き、初期化が終わったら左メニューの**イメージレポジトリ**をクリックし、AMMDS のイメージを検索します。

2. **イメージを選択**

   ダウンロード数が最も多いイメージを選びます。完全なイメージ名は `qyg2297248353/ammds` です。

![AMMDS Docker 镜像版本选择](/img/deploy/install-fnos-docker-01.png)

3. **バージョンを選択**

   デフォルトバージョンは `latest` です。お使いのデバイスが古い場合、`latest` をサポートしていない可能性があります。その場合はタグの末尾が `ol8` のバージョンを選んでください。

![AMMDS Docker 镜像下载完成](/img/deploy/install-fnos-docker-02.png)

4. **ダウンロード完了を待つ**

   ローカルイメージでダウンロードの進捗を確認できます。ダウンロードが完了すると実行ボタンが表示されます。

### コンテナの作成と設定

![AMMDS Docker 容器创建](/img/deploy/install-fnos-docker-03.png)

1. **コンテナを作成**

   **実行**ボタンをクリックしてデプロイを開始します。
   コンテナ名はお好みで（例：`ammds`）。
   デバイスの性能に合わせて適切なリソースを割り当てます。
   **起動時に自動実行**をオンにできます。

![AMMDS Docker 端口设置](/img/deploy/install-fnos-docker-04.png)

2. **ポート設定**

   詳細設定でポートマッピングを設定します：
   - デフォルトのコンテナポートは 80（TCP）
   - アクセスポートを 9523（または他の使用可能なポート）に変更

![AMMDS Docker 存储设置](/img/deploy/install-fnos-docker-05.png)

3. **ストレージ設定**

   ローカルディレクトリをコンテナにマウントすることをおすすめします。これでデータが失われません：
   - デフォルトでは飛牛が一時ストレージディレクトリを作成しますが、システムに削除される可能性があります
   - 対応するディレクトリを手動で作成し、コンテナにマウントすることを推奨します

![AMMDS Docker 信息确认](/img/deploy/install-fnos-docker-06.png)

4. **設定を確認**

   すべての設定が正しいことを確認したら、**作成後に起動**にチェックを入れます。

### 実行と管理

![AMMDS Docker 容器运行状态](/img/deploy/install-fnos-docker-07.png)

1. **実行状態を確認**

   コンテナ一覧で AMMDS が実行中であることを確認できます。右側の三点リーダーですばやく操作できます。

![AMMDS Docker 应用详情](/img/deploy/install-fnos-docker-08.png)

2. **アプリの詳細を確認**

   コンテナをクリックして詳細ページに入ります。

![AMMDS Docker 运行日志](/img/deploy/install-fnos-docker-09.png)

3. **実行ログを確認**

   **ログを表示**をクリックして実行状態やダウンロードの進捗を確認できます。

   :::info 初回起動のヒント
   初回インストール時、システムは必要なリソースファイルを自動ダウンロードします。しばらくお待ちください。ログにダウンロードの進捗が表示されます。
   :::

## インストール完了

![AMMDS 安装完成界面](/img/deploy/install-fnos-finish.png)

インストールが完了したら、ブラウザから AMMDS 管理画面にアクセスして使い始められます。

## 警告と注意事項

### セキュリティ警告

:::danger 重要なセキュリティのヒント

- **公式ソース**：AMMDS インストールパッケージは公式 GitHub Releases ページからのみダウンロードしてください。サードパーティの配布元は使わないでください
- **パスワードのセキュリティ**：強力なパスワードを設定し、定期的に変更しましょう
- **ポートのセキュリティ**：デフォルトポートを変更し、攻撃されるリスクを減らしましょう

:::

### システム要件

:::info システム構成要件

#### 最小要件

- オペレーティングシステム：飛牛 FnOS 最新バージョン
- プロセッサ：Intel Core i3 または同等のパフォーマンス
- メモリ：4GB RAM
- ディスク容量：少なくとも 10GB の空き容量
- ネットワーク：ブロードバンドインターネット接続

#### 推奨要件

- オペレーティングシステム：飛牛 FnOS 最新バージョン
- プロセッサ：Intel Core i5 または同等のパフォーマンス
- メモリ：8GB RAM 以上
- ディスク容量：20GB の空き容量
- ネットワーク：安定したブロードバンドインターネット接続

:::

### 使用上の注意事項

- **データバックアップ**：設定ファイルと重要なデータは定期的にバックアップしましょう
- **リソース管理**：ディスク容量に注意し、不要なファイルはこまめに削除しましょう
- **ネットワーク接続**：リソースダウンロードやアップデート時は、安定したネットワークを確保してください
- **システムメンテナンス**：飛牛 FnOS システムは定期的に更新しましょう

### トラブルシューティング

#### よくある問題

- **インストール失敗**：ディスク容量とネットワーク接続を確認してください
- **起動異常**：実行ログを確認してエラー情報を取得してください
- **アクセス失敗**：ポート設定とファイアウォール設定を確認してください
- **アップグレード失敗**：旧バージョンをアンインストールしてから新バージョンをインストールしてみてください

#### 連絡先

解決できない問題が発生した場合は、公式 GitHub リポジトリに Issue を提出してください。

## ベストプラクティス

- **定期更新**：AMMDS は常に最新バージョンを保ちましょう
- **適切な設定**：デバイスの性能に合わせてパラメータを適切に設定しましょう
- **安全な使用**：公式ドキュメントの手順に従い、コア設定ファイルはむやみに編集しないでください
- **コミュニティへの参加**：コミュニティに参加して、経験を共有したりフィードバックを送ったりしましょう
