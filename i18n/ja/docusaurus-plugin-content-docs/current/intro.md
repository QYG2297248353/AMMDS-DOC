# 概要

AMMDS (Adult Movie MetaData Scraper) は、あなたの個人用ムービーライブラリを簡単に管理できるプライベートデータプラットフォームです。

:::warning 注意
このソフトウェアはオフライン製品で、データはユーザーが手動で入力します。中国国内でこの製品を宣伝しないでください。必要に応じて略称「AMMDS」を使用してください。
:::

:::note プラグインについて
統合プラグインを使用するにはネットワーク接続が必要です。いずれかの統合プラグインを有効にすると、ネットワーク接続へのアクセスに同意したものとみなされます。
:::

## 私たちについて

私たちはテクノロジーと映画を愛する仲間です。メンバーはソフトウェア開発の経験が豊富で、映画やテレビドラマにも情熱を持っています。そんな私たちが AMMDS というプラットフォームを作りました。あなたの個人メディアライブラリを管理し、楽しむためにあります。

## プロジェクトの成り立ち

しばらく探していたのですが、市場には映画情報を自動で整理し、表紙やあらすじを補完し、複数のデバイスからアクセスできる便利なツールがありませんでした。そこで、自分たちで AMMDS を作ることにしました。あなたのパソコン上の動画ファイルとネット上の作品情報を自動で照合し、手作業で整理する手間を省きます。

## 機能の特徴

### 🎬 ムービーライブラリ管理
あなたのムービーライブラリをかんたん管理。複数の動画形式に対応し、作品情報を自動取得。あなただけのムービーライブラリを構築できます。

### 👤 自動俳優認識
作品内の俳優を自動認識し、詳細なプロフィールと作品リストを表示。検索や閲覧が簡単になります。

### 📄 メタデータの自動取得
インターネットから映画やテレビドラマの表紙、あらすじ、評価などの情報を自動取得。ライブラリをより豊かにします。

### 📤 共有機能
友人や家族とムービーライブラリを共有できます。ホームシアター体験を家族みんなで楽しめます。

### 🔌 プラグイン統合
さまざまなプラグインをインストール可能。お使いのメディアサーバーやプレイヤーと簡単に連携でき、機能を拡張できます。

### 🌐 複数ソースからの取得
TMDB、豆瓣など複数のデータソースから作品情報を取得。データの網羅性と正確性を両立します。

### 🤖 スマートマッチング
スマートアルゴリズムでローカルファイルとオンラインの作品情報を自動的に照合。手動操作を減らし、時短にもなります。

### ⚡ バッチ処理
複数の動画ファイルを一度に処理可能。管理効率がぐっと上がります。

### 🏷️ カスタムタグ
カスタムタグに対応。映画リソースを自分好みに分類・管理できます。

### 🖥️ クロスプラットフォーム対応
Windows、macOS、Linux など複数のプラットフォームで動作。いつでもどこでもムービーライブラリを管理できます。
