# 宝塔パネル

宝塔(bt.cn)は、運用管理を簡単かつ効率的にします。パネルは Linux と Windows システムをサポートしています。ワンクリック設定：LAMP/LNMP、Web サイト、データベース、FTP、SSL、Web インターフェースを通じてサーバーを簡単に管理します。
