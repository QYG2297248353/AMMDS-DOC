# スマート翻訳

スマート翻訳プラグインは、AMMDSプラットフォームに専門的な多言語翻訳機能を提供し、ユーザーが異なる言語間で正確な翻訳を実行できるようにサポートします。このプラグインは、高度な機械翻訳とAI翻訳技術を統合して、ユーザーに高品質で文脈正確な翻訳サービスを提供し、多様な翻訳ニーズを満たします。

このプラグインはシステムレベルの翻訳ソリューションを採用し、複数の翻訳エンジンとAIモデルをサポートしています。ユーザーは具体的なシナリオに応じて最適な翻訳方法を選択でき、翻訳品質と正確性を保証できます。

## プラグイン設定

![スマート翻訳プラグイン設定](/img/plugin/translation-01.png)

### 基本設定項目

- **翻訳を有効にする**: スマート翻訳機能を有効にするかどうかを制御します。有効にすると、システムは設定された翻訳サービスに従って翻訳リクエストを処理します。
- **AI優先**: AI翻訳サービスを優先的に使用するかどうかを設定します。AI翻訳が利用できない場合、システムは自動的に機械翻訳に切り替わり、翻訳サービスの継続性を保証します。
- **デフォルトの翻訳方法**: システムはデフォルトでMetaTube翻訳を使用します。ユーザーは必要に応じてスマート翻訳機能を有効にするかどうかを選択できます。

### 機械翻訳の追加

![機械翻訳の追加](/img/plugin/translation-02.png)

ユーザーは複数の機械翻訳サービスを設定して翻訳サービスクラスタを構築し、翻訳の信頼性と多様性を高めることができます：

- **プロバイダー名**: 翻訳エンジンに識別名を設定し、ユーザーが異なる翻訳サービスを管理・識別しやすくします。
- **翻訳プロバイダー**: Google翻訳、百度翻訳、テンセント翻訳などの異なる翻訳サービスプロバイダーを選択します。
- **優先度**: 翻訳エンジンの優先順位を設定します。システムは優先度に従って翻訳サービスを使用します。高優先度のサービスが利用できない場合、自動的に低優先度のサービスに切り替わります。

> 注意：異なる翻訳プロバイダーを選択すると、システムが対応する設定項目を表示します。ユーザーは実際のニーズに従ってAPIキーなどの必要な情報を入力する必要があります。

### AI翻訳の追加

![AI翻訳の追加](/img/plugin/translation-03.png)

ユーザーはAI翻訳サービスを設定し、高度な大規模言語モデルを活用して、より自然で正確な翻訳結果を提供できます：

- **プロバイダー名**: AI翻訳サービスに識別名を設定し、管理と識別を容易にします。
- **サービスタイプ**: OpenAIエンドポイントとGeminiエンドポイントの2つのサービスタイプをサポートし、ユーザーは実際の状況に応じて選択できます。
- **サービスアドレス**: AI翻訳サービスのAPIアドレスを入力し、公式APIアドレスまたは独自構築のプロキシアドレスをサポートします。
- **APIキー**: AI翻訳サービスにアクセスするためのキーを入力し、サービス呼び出しのセキュリティを保証します。
- **モデル**: 使用するAIモデルを選択します。異なるモデルは翻訳品質と速度に差がある場合があります。
- **温度**: AI翻訳の温度パラメータ（0-1の間）を設定し、翻訳結果の創造性を調整します。温度が低いほど翻訳結果は保守的で正確になり、温度が高いほど翻訳結果は柔軟で多様になります。

### AI翻訳プロンプトの設定

![AI翻訳プロンプトの設定](/img/plugin/translation-04.png)

ユーザーは具体的な翻訳シナリオに応じてAI翻訳プロンプトを設定し、翻訳結果を最適化して異なる分野の専門ニーズに適応できます：

- **プロンプト**: AI翻訳をガイドするためのプロンプトテキストを設定し、翻訳ニーズに応じてカスタマイズできます。例えば：
  - 専門分野翻訳：「以下のテキストを専門的な技術文書スタイルに翻訳し、用語の正確性を維持してください。」
  - 文学作品翻訳：「以下のテキストを流暢な文学スタイルに翻訳し、原文の感情と情趣を維持してください。」
  - ビジネス文書翻訳：「以下のテキストを正式なビジネススタイルに翻訳し、専門用語の正しい使用を保証してください。」

### 翻訳テスト

![翻訳テスト](/img/plugin/translation-05.png)

設定を完了した後、ユーザーは翻訳テストインターフェースでテキストを入力して翻訳結果をリアルタイムで確認し、翻訳効果を検証できます。テスト機能は以下をサポートします：
- 多言語間翻訳テスト
- 異なる翻訳エンジンの比較テスト
- 翻訳速度と品質の評価

## サポートされている言語

スマート翻訳プラグインは、以下を含む複数の一般的に使用される言語の相互翻訳をサポートします：
- 中国語（簡体字/繁体字）
- 英語
- 日本語
- 韓国語
- フランス語
- ドイツ語
- スペイン語
- ロシア語
- アラビア語

具体的にサポートされる言語の範囲は、設定された翻訳サービスプロバイダーによって異なります。

## 使用シナリオ

スマート翻訳プラグインは以下のシナリオに適しています：

1. **多言語コンテンツ管理**: AMMDSプラットフォームの多言語コンテンツに翻訳サポートを提供し、異なる言語バージョン間のコンテンツの一貫性を保証します。
2. **国際化アプリケーション**: 開発者がアプリケーションインターフェースとドキュメントを複数の言語に翻訳して、グローバルユーザーエクスペリエンスを向上させるのを支援します。
3. **言語間コミュニケーション**: 多言語環境でコミュニケーションコンテンツをリアルタイムで翻訳し、言語障壁を排除します。
4. **専門文書翻訳**: 技術文書、ビジネス文書などに専門的な翻訳サービスを提供し、用語の正確性を保証します。
5. **コンテンツローカライゼーション**: コンテンツを特定の言語と文化的背景に適応させ、ターゲットユーザーの受容度を高めます。

## ベストプラクティス

1. **翻訳サービスの混合使用**: 翻訳コンテンツのタイプと重要性に応じて適切な翻訳サービスを選択します。専門用語が多いコンテンツの場合、AI翻訳を使用することをお勧めします。日常的なコンテンツの場合、速度を向上させるために機械翻訳を使用できます。
2. **翻訳サービスの優先度の合理的な設定**: 信頼性の高い翻訳サービスを高優先度に設定し、翻訳サービスの安定性を保証します。
3. **AI翻訳プロンプトの最適化**: 異なる翻訳シナリオに応じて適切なプロンプトをカスタマイズし、翻訳品質を向上させます。
4. **定期的な翻訳サービステスト**: 設定された翻訳サービスを定期的にテストし、サービスの可用性と翻訳品質を保証します。
5. **翻訳使用状況の監視**: 翻訳サービスの使用状況を監視し、問題を及时发现して解決し、翻訳設定を最適化します。

## トラブルシューティング

1. **翻訳サービスが利用できない**: 翻訳サービスのAPIキーが正しいか、ネットワーク接続が正常か、サービスプロバイダーにサービス中断があるかを確認します。
2. **翻訳品質が低い**: AI翻訳の温度パラメータを調整し、プロンプトを最適化するか、異なる翻訳サービスを使用してみます。
3. **翻訳速度が遅い**: ネットワーク接続を確認し、単一翻訳のテキスト長を短縮するか、応答速度がより速い翻訳サービスを選択します。
4. **翻訳結果が不一致**: 同じ翻訳サービスと設定を使用しているかを確認し、異なるシナリオで異なる翻訳パラメータを使用しないようにします。
