# M-Team

M-Teamは多様な討論板とグループを持つ総合討論フォーラムで、世界の時事問題と情報の討論を提供しています。

:::info プライベートサイトについて
このサイトは招待制で登録が必要で、一定の要件があります。
:::

<!-- truncate -->

## プラグイン設定

![プラグイン設定](/img/plugin/m-team-01.png)

### 基本設定

- **有効状態**：M-Teamプラグインの有効化を制御します。
- **サービスアドレス**：M-Team APIサーバーアドレスです。デフォルト値は `https://api.m-team.io/api` で、実際の状況に応じて変更できます。
  
  現在のM-Team APIサーバーアドレスは以下の3つです：
  - https://api.m-team.cc/api
  - https://api.m-team.io/api
  - https://api2.m-team.cc/api
- **キー（Passkey）**：認証に使用するキーで、「コンソール >> 設定 >> ホーム >> キー >> メールで送信」ボタンから取得し、メールで結果を確認してください。デフォルト値は空です。
- **アクセストークン（Token）**：APIアクセスに使用するトークンで、「コンソール >> 設定 >> 実験室 >> アクセストークン >> アクセストークンの作成」ボタンで作成できます。デフォルト値は空です。

![アクセストークン](/img/plugin/m-team-02.png)

- **アカウント保護通知**：アカウントが14日間以上再ログインされない場合、システムが通知を送信し、アカウントが削除されるのを防ぐために再ログインする必要があります。
