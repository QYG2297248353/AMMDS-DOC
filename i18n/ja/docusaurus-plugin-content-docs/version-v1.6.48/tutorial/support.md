# サポートする

🥳 一緒に成長しましょう！

AMMDS サポートページへようこそ！ご関心とご支援をありがとうございます。あなたからのサポートは、AMMDS の機能とサービスの改善と拡張に直接役立ち、あなたや他のユーザーにより良い体験を提供することができます。

## どのようにサポートできますか？

### 1. 会員になる

オンラインストアを訪れて、AMMDS 関連の製品やサービスを購入してください。これにより、高品質な製品を獲得できるだけでなく、私たちの仕事を大いにサポートすることができます。

### 2. 友達にシェアする

AMMDS の使用が気に入ったら、ぜひご家族やお友達に紹介してください！シェアすることは私たちへの励みであるだけでなく、より多くの人々にこのアプリケーションを発見して使用してもらうのに役立ちます。

### 3. データの貢献

AMMDS プロジェクトのデータは共有モデルをサポートしています。これは、誰でもデータを表示、変更、改善の提案をすることができることを意味します。他の問題がある場合は、GitHub で遭遇した問題を報告してください。

### 4. フィードバックを提供する

あなたの意見は私たちにとって非常に重要です。機能のリクエスト、エラーレポート、その他のアイデアなど、いつでもお知らせください。公式ウェブサイトからお問い合わせいただくか、直接 GitHub で問題を提出することができます。

### 5. スポンサーになる

私たちの仕事が気に入って、継続的な発展をサポートしたい場合は、プロジェクトへの寄付を検討してください。あなたのサポートは、サーバーコスト、ドメイン費用、その他の必要な経費の支払いに役立ちます。

## 私たちの約束

私たちは、高品質な製品とサービスを提供することに常に取り組んでいます。私たちのチームは、あなたのニーズを満たし、より良い体験をもたらすために努力し続けます。

AMMDS は、映画とテレビ番組の愛好家のために設計されたデータ管理プラットフォームです。个人の映画・テレビ番組ライブラリを簡単に管理して楽しむのに役立ちます。メタデータの自動取得からローカルファイルのスマートマッチング、クロスプラットフォームサポートまで、AMMDS は最高のユーザーエクスペリエンスを提供することを目的としています。
