# 概要

AMMDS (Adult Movie MetaData Scraper) は、個人のメディアライブラリを管理して楽しむためのプライベートデータプラットフォームです。

<!-- truncate -->

:::warning 注意
このソフトウェアはオフライン製品であり、データはユーザーが手動で入力します。中国国内でこの製品を宣伝しないでください。必要に応じて略語：`AMMDS` を使用してください。
:::

:::note プラグインの注意
統合プラグインにはネットワーク機能が必要です。統合プラグインを有効にすると、ネットワーク接続へのアクセスに同意したものとみなされます。
:::

## 私たちについて

私たちは技術とイノベーションを愛する人々のグループです。私たちのチームメンバーは深いソフトウェアエンジニアリングのバックグラウンドを持つだけでなく、映画やテレビ番組にも独特の情熱を持っています。これが私たちに AMMDS を作成する啓発となりました。AMMDS は、個人のメディアライブラリを管理して楽しむために特別に設計されたプラットフォームです。

## プロジェクトの起源

ある日、映画やテレビ番組をよりよく保存、共有、整理、収集する方法について話し合っていると、自動的に整理し、メタデータを豊富にし、マルチプラットフォームアクセスをサポートするアプリケーションが市場に不足していることに気づきました。そこで、ユーザーがメディアコレクションを簡単に管理するだけでなく、スマートアルゴリズムを通じてローカルファイルとオンラインメタデータを自動的に一致させ、整理効率を大幅に向上させるツールを開発することに決めました。

## 機能特性

AMMDS は、メディアライブラリ管理体験を向上させるための包括的な機能セットを提供します：

### 🎬 映画ライブラリ管理
個人のメディアライブラリを簡単に管理し、複数のビデオ形式をサポートし、メタデータを自動的に取得し、専用のメディアライブラリを作成します。

### 👤 自動俳優認識
映画やテレビ番組の俳優情報を自動的に認識し、詳細な俳優プロフィールと作品リストを提供し、検索と閲覧を簡単にします。

### 📄 自動メタデータ取得
インターネットから映画やテレビ番組のメタデータ（表紙、紹介、評価など）を自動的に取得し、メディアライブラリの情報を豊富にします。

### 📤 共有機能
友人や家族とメディアライブラリを共有することをサポートし、メディアコレクションを簡単に共有し、家庭用シアター体験を実現します。

### 🔌 プラグイン統合
豊富なプラグインサポートを提供し、既存のメディアサーバーやプレーヤーに簡単に統合し、より多くの機能を拡張します。

### 🌐 複数ソース取得
複数のデータソース（TMDB、豆瓣など）から映画やテレビ番組の情報を取得することをサポートし、データの包括性と正確性を確保します。

### 🤖 スマートマッチング
スマートアルゴリズムを通じてローカルファイルとオンラインメタデータを自動的に一致させ、手動操作を減らし、整理効率を向上させます。

### ⚡ バッチ処理
複数のビデオファイルのバッチ処理をサポートし、管理効率を向上させます。

### 🏷️ カスタムタグ
カスタムタグをサポートし、映画リソースの個別分類と管理を行います。

### 🖥️ クロスプラットフォームサポート
Windows、macOS、Linux などの複数のプラットフォームでの実行をサポートし、いつでもどこでもメディアライブラリを管理します。
