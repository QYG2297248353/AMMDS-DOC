# Fanza DMM

FANZA は日本をリードする成人コンテンツプラットフォームで、オンライン DVD 販売、動画レンタル、ストリーミングサービスを提供しています。プラットフォームは複数の有名な動画制作会社と提携し、専属チャンネルサービスを提供するとともに、サブスクリプションベースの「見放題 ch」（見放題）サービスを提供しており、プラットフォーム内のほとんどの動画コンテンツをカバーしています。

## プラグイン設定

![プラグイン設定](/img/plugin/fanzadmm-01.png)

### 基本設定

| パラメータ | 説明 | デフォルト値 |
|------|------|--------|
| 有効状態 | Fanza DMM プラグインの有効化を制御 | オフ |
| Ranking ランキングデータ | FANZA のランキングデータの収集を制御 | オフ |
| 動画検索 | FANZA の動画コンテンツ検索機能の有効化を制御 | オフ |

### 設定の説明

- **ランキングデータ収集**：有効にすると、FANZA の各カテゴリのランキングデータを定期的に同期し、メディアライブラリに人気コンテンツの参考を提供します
- **動画検索**：有効にすると、FANZA データソースを通じて動画コンテンツを検索でき、メタデータのマッチング精度が向上します
- **データ更新頻度**：システムは設定に応じてデータ更新周期を自動的に調整し、データの最新性を確保します

## 機能特性

- **多次元データ**：動画、俳優、ランキングなどの多次元データをカバー
- **リアルタイム同期**：プラットフォームの最新データを定期的に同期することをサポート
- **高精度マッチング**：FANZA 公式データに基づく高精度なメタデータマッチング

## 注意事項

:::info 地域制限
FANZA サービスには地域アクセス制限が存在する場合があります。デプロイ時にはネットワーク環境の要因を考慮することを推奨します
:::

:::note データソース
プラグインのデータは FANZA 公式 API から取得されており、データの合法性と正確性が保証されています
:::
