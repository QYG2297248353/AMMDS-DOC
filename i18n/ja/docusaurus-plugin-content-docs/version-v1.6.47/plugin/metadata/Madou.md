# 麻豆传媒映画

麻豆传媒映画は、成人向けコンテンツの制作と配布に特化したメディア会社であり、米国カリフォルニア州に登録されており、サーバーインフラストラクチャは米国内に配置されています。このプラグインは、AMMDS に麻豆传媒の動画と俳優のメタデータ収集機能を提供します。

## プラグイン設定

![プラグイン設定](/img/plugin/madou-01.png)

### 基本設定

| パラメータ | 説明 | デフォルト値 |
|------|------|--------|
| 有効状態 | 麻豆传媒映画プラグインの有効化を制御 | オフ |
| サービスアドレス | 麻豆传媒の API サービスエンドポイント | `https://api.modelmediaasia.com` |
| 動画収集 | 麻豆传媒の動画メタデータの収集を制御（事前キャッシュメカニズムを採用） | オフ |
| 俳優収集 | 麻豆传媒の俳優メタデータの収集を制御（事前キャッシュメカニズムを採用） | オフ |

### 設定の説明

- **サービスアドレス**：デフォルトで公式 API エンドポイントを指しています。プロキシまたは自前のサービスを使用する場合は、ここで変更できます
- **メタデータ収集**：事前キャッシュメカニズムを採用しており、データ取得の速度と安定性を確保しています
- **データ同期**：収集を有効にすると、システムは最新の動画と俳優の情報を定期的に同期します

## 注意事項

:::note データ収集メカニズム
現在、プラグインはメタデータを事前にキャッシュする方式を採用しており、ネットワーク環境が不安定な場合でも完全なメタデータサービスを提供できるようになっています。
:::

:::info サービスの可用性
サービスが米国に展開されているため、ネットワーク接続の安定性を確保することを推奨します。必要に応じて、アクセス速度を最適化するためにプロキシを設定できます。
:::
