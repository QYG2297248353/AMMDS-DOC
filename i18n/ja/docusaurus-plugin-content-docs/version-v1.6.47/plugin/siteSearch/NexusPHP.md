# NexusPHP

NexusPHPは浙江大学のNexusチームによって開始・開発された、完全で秩序正しく、ユーザーの信頼性と知識を重視したリソース共有コミュニティソリューションを提供するPT（Private Tracker、プライベートトラッカーサーバー、BTダウンロードの一種）オープンソースソフトウェアです。

:::info プライベートサイトの説明
このサイトは招待制で登録が必要で、一定の要件があります。
:::

## プラグイン設定

![プラグイン設定](/img/plugin/nexusphp-01.png)

### 基本設定

- **有効状態**：NexusPHPプラグインの有効化を制御します。
- **検索機能**：NexusPHPの検索機能の有効化を制御します。デフォルト値は無効状態です。
- **ユーザー情報の同期**：NexusPHPのユーザー情報同期機能の有効化を制御します。デフォルト値は無効状態です。ユーザー情報はサイトランキングの表示に使用されます。
- **アラート通知**：NexusPHPのアラート通知機能の有効化を制御します。デフォルト値は無効状態です。

![アラート通知](/img/plugin/nexusphp-02.png)

## NexusPHPサイトモジュール

右上隅の「サイトを追加」ボタンをクリックして、サイトを追加します。

:::warning バージョン要件
NexusPHPサイトのバージョンは1.9.0以上である必要があります。

**バージョン確認方法**：NexusPHPサイトページの下部にある `Powered by NexusPHP` の箇所で、NexusPHPをクリックすることでサイトのバージョンを確認できます。
:::

![サイトを追加](/img/plugin/nexusphp-03.png)

### サイト設定項目

- **有効化**：このサイトを有効化するかどうか。デフォルト値は無効状態です。
- **サイト名**：プラグイン内に表示するためのサイト名。手動入力のほか、ドロップダウンメニューから一般的なサイト名を選択することで、システムが自動的にサイトアドレスを入力します。
- **サイトアドレス**：サイトにアクセスするためのアドレス。デフォルト値は空です。

![サイトトークン](/img/plugin/nexusphp-04.png)

- **サイトトークン**：「コントロールパネル >> 設定ホーム >> アクセストークン」で作成ボタンをクリックして作成します。

:::info 権限要件
**デフォルト権限**：トレントリストの取得、トレント詳細の表示、トレントの投稿、ユーザー基本情報の表示

**最低権限要件**：トレントリストの取得、トレント詳細の表示、ユーザー基本情報の表示

**権限不足**：権限が不足している場合、システムが `権限不足` と表示し、このサイトを正常に追加することができません。
:::

- **サイトロゴ**：プラグイン内に表示するためのサイトロゴ。デフォルト値は空です。
