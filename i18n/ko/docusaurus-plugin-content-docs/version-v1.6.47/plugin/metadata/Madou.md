# 麻豆传媒映画

麻豆传媒映画은 성인 콘텐츠 제작 및 배포에 전념하는 미디어 회사로，미국 캘리포니아주에 등록되어 있으며 서비스 인프라는 미국 내에 배포되어 있습니다。이 플러그인은 AMMDS에 마두 미디어의 영화 및 배우 메타데이터 수집 기능을 제공합니다。

## 플러그인 설정

![플러그인 설정](/img/plugin/madou-01.png)

### 기본 설정

| 매개변수 | 설명 | 기본값 |
|------|------|--------|
| 활성 상태 | 마두 미디어 영화 플러그인 활성화 여부를 제어 | 끔 |
| 서비스 주소 | 마두 미디어의 API 서비스 엔드포인트 | `https://api.modelmediaasia.com` |
| 영화 수집 | 마두 미디어의 영화 메타데이터 수집 여부를 제어（사전 캐싱 메커니즘 채택） | 끔 |
| 배우 수집 | 마두 미디어의 배우 메타데이터 수집 여부를 제어（사전 캐싱 메커니즘 채택） | 끔 |

### 설정 설명

- **서비스 주소**：기본적으로 공식 API 엔드포인트를 가리킵니다。프록시 또는 자체 서비스를 사용하려면 여기에서 수정할 수 있습니다
- **메타데이터 수집**：사전 캐싱 메커니즘을 채택하여 데이터 획득 속도와 안정성을 보장합니다
- **데이터 동기화**：수집을 활성화하면 시스템은 최신 영화 및 배우 정보를 정기적으로 동기화합니다

## 주의 사항

:::note 데이터 수집 메커니즘
현재 플러그인은 메타데이터를 사전 캐싱하는 방식을 채택하여 네트워크 환경이 불안정한 경우에도 완전한 메타데이터 서비스를 제공할 수 있도록 되어 있습니다。
:::

:::info 서비스 가용성
서비스가 미국에 배포되어 있으므로 네트워크 연결 안정성을 확보하는 것이 좋습니다。필요한 경우 접속 속도를 최적화하기 위해 프록시를 구성할 수 있습니다。
:::
