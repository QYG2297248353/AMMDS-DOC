# Fanza DMM

FANZA는 일본을 선도하는 성인 콘텐츠 플랫폼으로，온라인 DVD 판매、영화 대여 및 스트리밍 서비스를 제공합니다。플랫폼은 다수의 유명 영화사와 제휴 관계를 맺어 전용 채널 서비스를 출시했으며，구독 기반의「見放題 ch」（마음대로 보기）서비스를 제공하여 플랫폼 내 대부분의 비디오 콘텐츠를 포괄합니다。

## 플러그인 설정

![플러그인 설정](/img/plugin/fanzadmm-01.png)

### 기본 설정

| 매개변수 | 설명 | 기본값 |
|------|------|--------|
| 활성 상태 | Fanza DMM 플러그인 활성화 여부를 제어 | 끔 |
| Ranking 랭킹 데이터 | FANZA의 랭킹 데이터 수집 여부를 제어 | 끔 |
| 영화 검색 | FANZA의 영화 콘텐츠 검색 기능 활성화 여부를 제어 | 끔 |

### 설정 설명

- **랭킹 데이터 수집**：활성화하면 FANZA 각 카테고리의 랭킹 데이터를 정기적으로 동기화하여 미디어 라이브러리에 인기 콘텐츠 참고를 제공합니다
- **영화 검색**：활성화하면 FANZA 데이터 소스를 통해 영화 콘텐츠를 검색할 수 있어 메타데이터 매칭 정확도가 향상됩니다
- **데이터 업데이트 빈도**：시스템은 설정에 따라 데이터 업데이트 주기를 자동으로 조정하여 데이터의 최신성을 보장합니다

## 기능特性

- **다차원 데이터**：영화、배우、랭킹 등 다차원 데이터를 포괄
- **실시간 동기화**：플랫폼의 최신 데이터를 정기적으로 동기화하는 것을 지원
- **정확한 매칭**：FANZA 공식 데이터 기반의 고정밀도 메타데이터 매칭

## 주의 사항

:::info 지역 제한
FANZA 서비스에는 지역 액세스 제한이 있을 수 있으므로 배포 시 네트워크 환경 요소를 고려하는 것이 좋습니다
:::

:::note 데이터 소스
플러그인 데이터는 FANZA 공식 API에서 가져오므로 데이터의 합법성과 정확성이 보장됩니다
:::
