# Emby

Emby는 디지털 미디어 파일의 정리, 관리, 공유를 전문으로 하는 강력한 멀티미디어 관리 애플리케이션 스위트입니다. 이 플러그인을 통해 AMMDS는 Emby 미디어 서버와의 통합을 구축하고 메타데이터 동기화 및 미디어 라이브러리 관리를 실현할 수 있습니다.

## 플러그인 설정

![플러그인 설정](/img/plugin/emby-01.png)

### 기본 설정

| 매개변수 | 설명 | 기본값 |
|------|------|--------|
| 활성 상태 | Emby 플러그인 활성화 여부를 제어 | 끔 |
| 서비스 주소 | Emby 서버의 네트워크 엔드포인트 | `http://127.0.0.1:8096` |
| API 키 | Emby 서버의 인증 토큰 | - |
| 사용자명 | (선택사항) Emby 서버의 로그인 사용자명 | - |
| 비밀번호 | (선택사항) Emby 서버의 로그인 비밀번호 | - |
| 사용자 바인딩 | 바인딩할 Emby 사용자 계정을 선택 | - |

![플러그인 설정](/img/plugin/emby-02.png)

### 설정 설명

- **서비스 주소**：서버의 실제 네트워크 주소를 사용하십시오。`localhost` 또는 `127.0.0.1` 사용은 피하십시오。그렇지 않으면 컨테이너화된 배포 환경에서 연결 오류가 발생할 수 있습니다
- **API 키**：Emby 관리 인터페이스의「시스템」→「고급」→「API 키」에서 생성할 수 있습니다
- **사용자 바인딩**：올바른 서비스 주소와 API 키를 입력하고 저장한 후에만 드롭다운 목록에서 사용자를 선택할 수 있습니다

### 연결 확인

- **네트워크 연결성**：AMMDS 서버가 Emby 서비스 주소에 액세스할 수 있는지 확인하십시오
- **인증 정보**：API 키에는 필요한 작업을 수행하기 위한 충분한 권한이 필요합니다
- **서비스 상태**：Emby 서버가 실행 중이고 API 요청에 정상적으로 응답할 수 있는지 확인하십시오

## 기능特性

- **메타데이터 동기화**：Emby 미디어 라이브러리와의 메타데이터 양방향 동기화를 지원
- **사용자 바인딩**：특정 Emby 사용자를 바인딩하여 개인화된 설정을 실현
- **상태 모니터링**：Emby 서버의 실행 상태를 실시간으로 모니터링

## 주의 사항

:::warning 서비스 주소 설정
특히 컨테이너화된 배포 환경에서는 `localhost` 또는 `127.0.0.1`을 서비스 주소로 사용하지 마십시오。그렇지 않으면 연결 오류가 발생할 수 있습니다。
:::

:::info API 키 권한
시스템 보안을 확보하기 위해 AMMDS용으로 전용 API 키를 생성하고 적절한 권한 범위를 부여하는 것이 좋습니다。
:::
