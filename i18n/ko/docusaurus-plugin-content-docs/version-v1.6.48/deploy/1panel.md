# 1Panel 배포 가이드

1Panel은 강력한 서버 관리 패널로서 직관적인 Web 인터페이스를 제공하여 사용자가 Linux 서버 내의 애플리케이션, 웹사이트, 파일, 데이터베이스 및 대형 언어 모델(LLM)과 같은 리소스를 효율적으로 관리하도록 돕습니다. 이 가이드에서는 1Panel 환경에서 AMMDS를 배포하는 단계에 대해 자세히 설명합니다.

<!-- truncate -->

## 타사 앱 스토어 설치

이 방법은 1Panel v1 및 v2 버전에 적용됩니다.

### 자동 설치

자동 설치 방법에서는 타사 앱 스토어의 내용이 3시간마다 자동으로 업데이트되어 항상 최신 앱 리소스를 얻을 수 있습니다.

#### 설치 스크립트

```bash
curl -sSL https://install.lifebus.top/auto_install.sh | bash
```

#### 제거 스크립트

```bash
curl -sSL https://install.lifebus.top/auto_uninstall.sh | bash
```

### 수동 설치

수동 설치 스크립트는 최신 타사 앱 스토어 내용을 한 번만 가져오며, 이후 스토어 내용 또는 버전 업데이트는 이 스크립트를 다시 실행해야 합니다.

```bash
curl -sSL https://install.lifebus.top/app_install.sh | bash
```

## 스토어 내용 업데이트

### 작업 단계

1. **스토어 설정에 들어가기**
   왼쪽 탐색 바에서 앱 스토어로 들어가 스토어 설정 페이지로 이동합니다.


2. **업데이트 작업 실행**

   ![更新商店内容](/img/deploy/install-1panel-00.png)
   
   #### v1 버전
   스토어 페이지의 오른쪽 상단에 있는「更新应用列表」버튼을 클릭합니다.

   ![更新商店内容](/img/deploy/install-1panel-01.png)

   #### v2 버전
   스토어 페이지의 왼쪽 상단에 있는「同步本地应用」버튼을 클릭합니다.

## 앱 설치

### 앱 검색 및 선택

![安装](/img/deploy/install-1panel-02.png)

1. **AMMDS 앱 검색**
   앱 스토어에서 "ammds" 앱을 검색합니다.

2. **버전 선택**
   시스템에 두 버전의 AMMDS 앱이 표시됩니다:
   - **호환 버전**: 하드웨어 구성이 낮은 장치에 적합합니다
   - **최신 버전**: 하드웨어 구성이 높은 장치에 적합합니다

   ![安装](/img/deploy/install-1panel-03.png)

### 설치 매개변수 구성

설치 버튼을 클릭하면 매개변수 구성 패널이 표시됩니다.

#### 기본 구성

- **이름**: "ammds"로 입력하는 것이 좋지만 필요에 따라 사용자 정의할 수도 있습니다
- **버전**: 최신 기능을 얻기 위해 "최신 버전"을 선택하는 것이 좋습니다
- **네트워크 모드**: 특별한 요구 사항이 없는 한 기본 구성을 유지하는 것이 좋습니다
- **데이터 영속성 경로**: AMMDS 앱 데이터를 저장하기 위한 서버의 절대 경로를 입력합니다
- **WebUI 포트**: 8080을 입력하는 것이 좋지만 필요에 따라 사용자 정의할 수도 있습니다
- **API 포트**: 8081을 입력하는 것이 좋지만 필요에 따라 사용자 정의할 수도 있습니다

다른 매개변수는 선택적 구성으로 실제 요구 사항에 따라 입력할 수 있습니다.

![安装](/img/deploy/install-1panel-04.png)

#### 마운트 디렉토리 설정

**사용자 정의 마운트 디렉토리**: 미디어 파일이 있는 디렉토리를 입력하는 것이 좋습니다(선택 사항).

![安装](/img/deploy/install-1panel-05.png)

#### 포트 액세스 구성

**「포트 외부 액세스」옵션을 반드시 체크하세요**. 그렇지 않으면 네트워크를 통해 WebUI와 API에 액세스할 수 없습니다.

「확인」버튼을 클릭하여 AMMDS 앱 설치를 시작합니다.

## 설치 완료

![安装完成](/img/deploy/install-1panel-06.png)

### 설치 성공 확인

설치가 완료되면「已安装」탭에서 설치된 AMMDS 앱을 확인할 수 있습니다.

### 관리 기능 소개

앱 카드에는 여러 가지 빠른 작업 버튼이 제공됩니다:
- **터미널**: 앱 컨테이너의 터미널에 액세스합니다
- **로그**: 앱 실행 로그를 표시합니다
- **시작**: 앱 서비스를 시작합니다
- **중지**: 앱 서비스를 중지합니다
- **재시작**: 앱 서비스를 재시작합니다
- **제거**: 앱을 삭제합니다
- **재구성**: 앱 컨테이너를 재구성합니다

## 앱에 액세스

![访问应用](/img/deploy/install-1panel-07.png)

### 액세스 방법

설치가 완료되면 브라우저를 사용하여 AMMDS의 WebUI와 API 인터페이스에 액세스할 수 있습니다:

- **WebUI 주소**: http://&lt;서버IP&gt;:&lt;WebUI포트&gt;
- **API 주소**: http://&lt;서버IP&gt;:&lt;API포트&gt;

### 기본 로그인 정보

- **사용자 이름**: ammds
- **비밀번호**: ammds

:::info 보안 팁
최초 로그인 후 시스템 보안을 위해 기본 비밀번호를 즉시 변경하세요.
:::

## 고급 시도

### 사용자 정의 구성

Docker 컨테이너 관리에 익숙한 경우 1Panel의 고급 구성 옵션을 통해 AMMDS 앱에 대해 보다 세밀한 설정을 할 수 있습니다:

- 리소스 제한 조정(CPU, 메모리)
- 환경 변수 구성
- 네트워크 고급 설정
- 스토리지 볼륨 관리

### 다중 인스턴스 배포

다른 기능 시나리오를 분리할 필요가 있는 사용자는 1Panel에서 여러 AMMDS 인스턴스를 배포하고 다른 포트와 데이터 경로로 구분할 수 있습니다.

## 경고 및 주의 사항

### 보안 경고

:::danger 중요 보안 팁

- **공식 출처**: 앱 설치 스크립트는 공식 채널에서만 가져오고 타사 다운로드 원본 사용을 피하세요
- **비밀번호 보안**: 강력한 비밀번호를 설정하고 정기적으로 변경하며 약한 비밀번호 사용을 피하세요
- **포트 보안**: 기본 포트를 변경하여 공격 위험을 줄이세요
- **외부 액세스**: 실제 요구에 따라 포트 외부 액세스를 구성하고 불필요한 노출을 피하세요

:::

### 시스템 요구 사항

:::info 시스템 구성 요구 사항

#### 최소 구성

- 운영 체제: Linux 배포판(1Panel 설치 지원)
- 프로세서: Intel Core i3 또는 동등한 성능
- 메모리: 4GB RAM
- 디스크 공간: 최소 10GB 사용 가능 공간
- 네트워크: 광대역 인터넷 연결

#### 권장 구성

- 운영 체제: Linux 배포판(1Panel 설치 지원)
- 프로세서: Intel Core i5 또는 동등한 성능
- 메모리: 8GB RAM 이상
- 디스크 공간: 20GB 사용 가능 공간
- 네트워크: 안정적인 광대역 인터넷 연결

:::

### 사용 주의 사항

- **데이터 백업**: 정기적으로 AMMDS의 구성 파일과 중요한 데이터를 백업하세요
- **리소스 관리**: 서버 리소스 사용 상태를 모니터링하고 리소스 과도 사용을 피하세요
- **네트워크 연결**: 특히 리소스 다운로드 및 업데이트 시 안정적인 네트워크 연결을 보장하세요
- **시스템 유지 관리**: 시스템 안정성을 보장하기 위해 정기적으로 1Panel 및 관련 앱을 업데이트하세요

### 문제 해결

#### 자주 발생하는 문제

- **설치 실패**: 네트워크 연결이 정상인지 디스크 공간이 충분한지 확인하세요
- **시작 이상**: 앱 로그를 확인하여 자세한 오류 정보를 얻으세요
- **액세스 실패**: 포트 구성이 올바른지 방화벽이 액세스를 허용하는지 확인하세요
- **업그레이드 실패**: 구 버전을 제거한 후 새 버전을 설치해 보세요

#### 연락처

해결할 수 없는 문제가 발생한 경우 공식 GitHub 리포지토리에 접속하여 Issue를 제출하거나 도움을 요청하세요.
