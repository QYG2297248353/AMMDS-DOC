# 소개

AMMDS (Adult Movie MetaData Scraper)는 개인 미디어 라이브러리를 관리하고 즐기기 위한 개인 데이터 플랫폼입니다.

<!-- truncate -->

:::warning 주의
이 소프트웨어는 오프라인 제품이며 데이터는 사용자가 수동으로 입력합니다. 중국 내에서 이 제품을 홍보하지 마십시오. 필요한 경우 약어 `AMMDS`를 사용하십시오.
:::

:::note 플러그인 알림
통합 플러그인에는 네트워크 기능이 필요합니다. 어떤 통합 플러그인이든 활성화하면 네트워크 연결에 대한 액세스에 동의한 것으로 간주됩니다.
:::

## 우리에 대하여

우리는 기술과 혁신을 사랑하는 사람들로 구성된 그룹입니다. 우리 팀 멤버는 깊은 소프트웨어 엔지니어링 배경을 가지고 있을 뿐만 아니라 영화와 TV 프로그램에 대한 독특한 열정을 가지고 있습니다. 이것이 우리를 AMMDS를 창조하도록 영감을 주었습니다. AMMDS는 개인 미디어 라이브러리를 관리하고 즐기기 위해 특별히 설계된 플랫폼입니다.

## 프로젝트의 기원

어느 날, 우리는 영화와 TV 프로그램을 더 잘 저장, 공유, 조직화 및 수집하는 방법에 대해 이야기하던 중 시장에는 자동으로 조직화하고 메타데이터를 풍부하게 하고 다중 플랫폼 액세스를 지원하는 애플리케이션이 부족하다는 것을 발견했습니다. 따라서 우리는 사용자가 미디어 컬렉션을 쉽게 관리할 수 있을 뿐만 아니라 지능형 알고리즘을 통해 로컬 파일과 온라인 메타데이터를 자동으로 일치시켜 조직화 효율성을 크게 향상시키는 도구를 개발하기로 결정했습니다.

## 기능 특징

AMMDS는 미디어 라이브러리 관리 경험을 향상시키기 위한 포괄적인 기능 세트를 제공합니다:

### 🎬 영화 라이브러리 관리
개인 미디어 라이브러리를 쉽게 관리하고, 다양한 비디오 형식을 지원하고, 메타데이터를 자동으로 가져오고, 전용 미디어 라이브러리를 만듭니다.

### 👤 자동 배우 식별
영화와 TV 프로그램의 배우 정보를 자동으로 식별하고, 자세한 배우 프로필과 작품 목록을 제공하여 찾기와 탐색을 쉽게 만듭니다.

### 📄 자동 메타데이터 가져오기
인터넷에서 영화와 TV 프로그램의 메타데이터(표지, 소개, 평점 등)를 자동으로 가져와 미디어 라이브러리 정보를 풍부하게 합니다.

### 📤 공유 기능
친구나 가족과 미디어 라이브러리를 공유하는 것을 지원하여 미디어 컬렉션을 쉽게 공유하고 홈 시어터 경험을 만듭니다.

### 🔌 플러그인 통합
풍부한 플러그인 지원을 제공하여 기존 미디어 서버나 플레이어에 쉽게 통합하고 더 많은 기능을 확장합니다.

### 🌐 다중 소스 가져오기
여러 데이터 소스(TMDB, 도우반 등)에서 영화와 TV 프로그램 정보를 가져오는 것을 지원하여 데이터의 포괄성과 정확성을 보장합니다.

### 🤖 지능형 매칭
지능형 알고리즘을 통해 로컬 파일과 온라인 메타데이터를 자동으로 일치시켜 수동 작업을 줄이고 조직화 효율성을 향상시킵니다.

### ⚡ 일괄 처리
여러 비디오 파일의 일괄 처리를 지원하여 관리 효율성을 향상시킵니다.

### 🏷️ 사용자 정의 태그
사용자 정의 태그를 지원하여 영화 리소스를 개인화된 방식으로 분류하고 관리합니다.

### 🖥️ 크로스 플랫폼 지원
Windows, macOS, Linux 등 여러 플랫폼에서 실행되는 것을 지원하여 언제 어디서나 미디어 라이브러리를 관리합니다.
