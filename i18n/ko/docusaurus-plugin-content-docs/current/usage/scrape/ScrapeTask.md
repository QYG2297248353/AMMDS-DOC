# 스크래핑 작업 실행

스크래핑 작업은 스크래핑 프로세스의 두 번째 단계로，스크래핑 규칙에 따라 스캔 결과 파일을 스크래핑하여 영상 메타데이터를获取하고 최종 영상 파일을 생성하여 영상整理 작업을完成합니다.

<!-- truncate -->

## 선행 조건

스크래핑 작업을 실행하기 전에 다음 조건이 충족되는지 확인하세요：

### 네트워크 연결

文件整理에는 인터넷을 통해 영상 메타데이터를获取해야 하므로 프로젝트가 인터넷에 정상적으로 액세스할 수 있는지确保해야 합니다.

### 메타데이터 플러그인

영상 메타데이터를获取하는 핵심인 관련 메타데이터 플러그인이 활성화되고正确하게 구성되어 있는지 확인하세요：

+ [Metatube 플러그인](/docs/plugin/metadata/Metatube)
+ [ThePornDB 플러그인](/docs/plugin/metadata/ThePornDB)
+ [StashBox 플러그인](/docs/plugin/metadata/ThePornDB)
+ [FanzaDMM 플러그인](/docs/plugin/metadata/FanzaDmm)
+ [Madou 플러그인](/docs/plugin/metadata/Madou)

:::warning 중요한 경고
관련 메타데이터 플러그인을 활성화하지 않으면 영상 메타데이터를获取할 수 없어 스크래핑이失败합니다.
:::

:::info 데이터 출처 선언
본 소프트웨어는 어떠한 소스 데이터도 제공하지 않으며，모든 영상 메타데이터는 사용자가 자체적으로 구성한 플러그인 서비스에서 제공됩니다.
:::

## 스크래핑 작업 실행

### 단계 1：스크래핑 및 입고 페이지로 이동

左侧导航栏을 통해 **작업 관리**로 들어간 후 **스크래핑 및 입고** 옵션을 선택하여 스크래핑 및 입고 페이지로 이동합니다.

![스크래핑 및 입고 페이지](/img/usage/scrape/task-scrape-01.png)

### 단계 2：스크래핑할 파일 선택

스크래핑 및 입고 페이지에서 스캔 작업을 통해记录된 모든 대기 중인 스크래핑 파일 목록을 볼 수 있습니다. 필요에 따라 스크래핑하려는 파일을 선택하세요.

### 단계 3：스크래핑 작업 실행 시작

페이지의 **스크래핑 및 정리** 버튼을 클릭하면 시스템이 스크래핑 작업을 실행하기 시작합니다.

![스크래핑 작업 실행](/img/usage/scrape/task-scrape-02.png)

### 단계 4：작업 실행 상태查看

작업 실행过程中，페이지에 스크래핑 작업的执行情况이实时으로 표시됩니다. 다음을 포함합니다：
- 스크래핑에 성공한 파일 수
- 스크래핑에失败한 파일 수
- 건너뛴 파일 수

### 단계 5：스크래핑 작업 종료（필요한 경우）

실행过程中，스크래핑 작업을 종료해야 하는 경우随时 **취소** 버튼을 클릭하여 작업 실행을 중지할 수 있습니다.

## 스크래핑 원리说明

스크래핑 작업의 작동 원리는 다음과 같습니다：

1. **스크래핑 규칙 읽기**：시스템은 구성한 스크래핑 규칙에 따라不同类型的文件을如何处理할지 결정합니다.
2. **메타데이터 매칭**：활성화된 메타데이터 플러그인을 통해 영상의详细信息를 매칭하고获取하려尝试합니다.
3. **영상 파일 생성**：获取한 메타데이터에 따라 시스템은标准化된 영상 파일을生成합니다.
4. **文件结构整理**：预设된命名 규칙과 디렉토리结构에 따라 영상 파일을整理합니다.

## 자주 발생하는 문제与解决方案

### 문제：스크래핑失败

**可能的原因**：
- 네트워크 연결问题
- 메타데이터 플러그인이 활성화되지 않았거나 구성错误
- 스크래핑 규칙设置不当
- 文件命名이 매칭 규칙을满足하지 않음

**解决方案**：
- 네트워크 연결이正常한지检查하세요
- 메타데이터 플러그인이 활성화되고正确하게 구성되었는지确认하세요
- 스크래핑 규칙设置이合理한지检查하세요
-文件命名格式을修改하여 매칭 규칙에 더符合하도록尝试하세요

### 문제：메타데이터获取不完整

**可能的原因**：
- 플러그인 서비스가暂时不可用
- 플러그인 서비스에 영상信息不存在
- 네트워크 지연导致获取超时

**解决方案**：
-稍後重试，플러그인 서비스가暂时不可用일 수 있습니다
-其他 메타데이터 플러그인을使用해 보세요
-네트워크 연결稳定性을检查하세요

## 모범 사례

1. **분할 처리**：大量文件의 경우 시스템负载를避免하기 위해 분할하여 스크래핑하는 것이 좋습니다
2. **우선 구성**：스크래핑을 실행하기 전에 스크래핑 규칙과 메타데이터 플러그인이正确하게 구성되어 있는지确保하세요
3. **定期更新**：更好的 매칭效果를 얻기 위해 메타데이터 플러그인을定期更新하세요
4. **로그检查**：스크래핑失败时详细错误信息를了解하기 위해系统 로그를查看하세요

## 주의事项

:::caution 주의
스크래핑 작업은文件数量와 네트워크 속도에 따라 다르지만 완료하는 데 시간이 걸릴 수 있으니耐心等待하세요.
:::

:::tip 팁
최佳 스크래핑效果를 얻기 위해文件命名을尽可能规范하게 유지하고 필요한识别信息（编号、제목等）를 포함하도록建议합니다.
:::
