# 구독 규칙 구성

구독 규칙 구성은 작업 실행 전 영상 콘텐츠의 구독 선호도와 다운로드 매개변수를 설정하여 시스템이 귀하의 요구에 따라 영상 자원을 자동으로获取하고 관리할 수 있도록 합니다.

<!-- truncate -->

## 구독 규칙 구성에 액세스

다음 경로를 통해 구독 규칙 구성 페이지에 액세스할 수 있습니다:

**작업 관리 >> 구독 규칙**

## 구성 항목 설명

### 중국어 우선

**기능 설명**: 이 옵션을 활성화하면 시스템은 중국어 자막 또는 중국어 더빙이 포함된 영상 버전을 우선적으로 검색하고 다운로드합니다.

**적용 시나리오**: 중국어 콘텐츠를 더 선호하는 경우 이 옵션을 활성화하여 중국어 버전을 얻을 확률을提高할 수 있습니다.

### 검색 순서

**기능 설명**: 검색 소스의 우선 순위를 설정합니다. 시스템은 지정된 순서대로 각 검색 소스에서 영상 자원을 차례로获取합니다.

**구성 권장 사항**: 자원 품질이 더 높고 업데이트 속도가 더 빠른 검색 소스를 앞에 배치하여 고품질 자원을 얻는 효율성을提高하세요.

### 다운로드 디렉토리

**기능 설명**: 다운로더의 다운로드 디렉토리 경로를 지정합니다.

**중요 안내**:

:::warning 주의사항
- 이 경로는 AMMDS 시스템 자체와 직접적인 관계가 없으며 사용하는 다운로더의 구성에 따라 설정해야 합니다
- 지정된 디렉토리가存在하고 쓰기 권한이 있는지 확인하세요
- 상대 경로로 인한 경로 해석 오류를 피하기 위해 절대 경로를 사용하는 것이 좋습니다
:::

## 구성 예제

다음은 전형적인 구독 규칙 구성 예제입니다:

1. **중국어 우선**: 활성화
2. **검색 순서**: 검색 소스 A → 검색 소스 B → 검색 소스 C
3. **다운로드 디렉토리**: `D:\Downloads\Movies`

## 자주 묻는 질문

### Q: 중국어 우선을 설정했지만 여전히 비중국어 버전을 다운로드했어요. 왜 그런가요?

**A**: 현재 영상 자원에 중국어 버전이 출시되지 않았기 때문일 수 있습니다. 시스템은 중국어 버전이 없는 경우 다른 버전을 다운로드하여 귀하가 자원을 즉시获取할 수 있도록 합니다.

### Q: 다운로드 디렉토리를 잘못 설정하면 어떤 문제가 발생하나요?

**A**: 다운로드 디렉토리를 잘못 설정하면 다음과 같은 문제가 발생할 수 있습니다:
- 다운로더가 파일을 정상적으로保存할 수 없음
- 시스템이 다운로드가 완료된 파일을 찾을 수 없음
- 저장 공간 부족으로 인한 다운로드 실패

올바른 다운로드 디렉토리 경로를 설정하는 것이 중요합니다.
