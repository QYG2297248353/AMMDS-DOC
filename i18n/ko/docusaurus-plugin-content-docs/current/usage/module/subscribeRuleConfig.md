# 구독 규칙 설정

구독 규칙 설정은 작업 실행 전에 비디오 콘텐츠의 구독 기호와 다운로드 매개변수를 설정하여 시스템이 귀하의 요구에 따라 자동으로 비디오 자원을获取하고 관리할 수 있도록 합니다.

<!-- truncate -->

## 구독 규칙 설정 접근

다음 경로를 통해 구독 규칙 설정 페이지에 접근할 수 있습니다：

**작업 관리 >> 구독 규칙**

## 설정 항목 설명

### 중국어 우선

**기능 설명**：이 옵션을 활성화하면 시스템이 중국어 자막 또는 중국어 더빙이 있는 비디오 버전을 우선적으로 검색하고 다운로드합니다.

**적용 시나리오**：중국어 콘텐츠를 더 선호하는 경우 이 옵션을 활성화하여 중국어 버전을 획득할 확률을 향상시킬 수 있습니다.

### 검색 순서

**기능 설명**：검색 소스의 우선 순위를 설정하면 시스템이 귀하가 지정한 순서대로 각 검색 소스에서 비디오 자원을 순차적으로获取합니다.

**구성 권장 사항**：자원 품질이 더 높고 업데이트 속도가 더 빠른 검색 소스를 앞에 배치하여 고품질 자원을 획득하는 효율성을 향상시켜세요.

### 다운로드 디렉토리

**기능 설명**：다운로더의 다운로드 디렉토리 경로를 지정합니다.

**중요한 참고 사항**：

:::warning 주의 사항
- 이 경로는 AMMDS 시스템 자체와 직접적인 관계가 없으며，사용하는 다운로더의 구성에 따라 설정해야 합니다
- 지정한 디렉토리가 존재하고 쓰기 권한이 있는지 확인하세요
- 상대 경로로 인한 경로 해석 오류를 방지하기 위해 절대 경로 사용을 권장합니다
:::

## 설정 예시

다음은 일반적인 구독 규칙 설정 예시입니다：

1. **중국어 우선**：활성화
2. **검색 순서**：검색 소스 A → 검색 소스 B → 검색 소스 C
3. **다운로드 디렉토리**：`D:\Downloads\Movies`

## 자주 묻는 질문

### Q: 중국어 우선을 설정했지만 여전히 비중국어 버전을 다운로드했습니다. 왜 그럴까요？

**A**：현재 비디오 자원에 중국어 버전이 출시되지 않았기 때문일 수 있습니다. 시스템은 중국어 버전이 없는 경우 다른 버전을 다운로드하여 귀하가 자원을 신속히 획득할 수 있도록 합니다.

### Q: 다운로드 디렉토리 설정 오류는 어떤 문제를 일으킬 수 있나요？

**A**：다운로드 디렉토리 설정이 잘못된 경우 다음과 같은 문제가 발생할 수 있습니다：
- 다운로더가 파일을 정상적으로 저장할 수 없음
- 시스템이 다운로드 완료된 파일을 찾을 수 없음
- 저장 공간이 부족할 때의 다운로드 실패

정확한 다운로드 디렉토리 경로를 설정했는지 확인하세요.
