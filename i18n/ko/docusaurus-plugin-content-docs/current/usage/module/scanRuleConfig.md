# 스캔 설정

스캔 설정은 작업 실행 전에 시스템이 미디어 파일을 어떻게 인식하고 처리할지 설정하여 스캔 프로세스가 귀하의 비디오 자원을 정확하고 효율적으로 인식할 수 있도록 합니다.

<!-- truncate -->

## 스캔 설정 접근

다음 경로를 통해 스캔 규칙 설정 페이지에 접근할 수 있습니다:

**작업 관리 >> 스캔 규칙 설정**

:::info 작업 팁
설정을 업데이트한 후 **저장** 버튼을 클릭해야 설정이 적용됩니다.
:::

## 설정 항목 설명

### 포맷 제한

**기능 설명**：시스템이 인식할 수 있는 미디어 파일 유형을 설정합니다.

**기본 지원**：시스템에는 `.mp4`, `.mkv`, `.avi`, `.mov` 등과 같은 일반적인 미디어 파일 유형이 기본적으로 내장되어 있습니다.

**사용자 정의 추가**：필요에 따라 다른 파일 유형을 추가할 수 있습니다. 추가할 때 파일 유형의 확장자(예: `.strm`)를 입력한 후 **Enter** 키를 눌러 입력을 확인하세요.

### 디렉토리 무시

**기능 설명**：스캔 시 무시할 디렉토리를 구성하면 해당 디렉토리 및 하위 디렉토리의 파일이 스캔되지 않습니다.

**구성 방법**：디렉토리의 상대 경로(예: `sample`, `tmp`)를 입력하세요. 디렉토리 이름은 대소문자를 구분하지 않습니다.

**적용 시나리오**：샘플, 임시 파일 또는 기타 비영상 자원이 포함된 디렉토리를 무시하는 데 사용할 수 있습니다.

### 파일 무시

**기능 설명**：스캔 시 무시할 파일을 구성하면 지정된 문자열을 포함하는 파일이 스캔되지 않습니다.

**구성 방법**：파일 이름에 포함된 문자열(예: `sample`, `hh800.com@`)을 입력하세요. 문자열은 대소문자를 구분하지 않습니다.

**적용 시나리오**：샘플 파일, 특정 마크가 있는 파일 또는 기타 처리할 필요가 없는 파일을 무시하는 데 사용할 수 있습니다.

### 분할 마크

**기능 설명**：분할 비디오 파일의 마크 문자열을 구성하여 시스템이 분할 비디오를 인식하고 개별 파일로 처리하지 않도록 도와줍니다.

**구성 방법**：분할 마크 문자열(예: `part`, `cd`)을 입력하세요. 문자열은 대소문자를 구분하지 않습니다.

**적용 시나리오**：여러 부분으로 분할된 비디오 파일을 처리하는 데 적합하며, 시스템이 이 파일들이 동일한 비디오 자원에 속하는 것을 올바르게 인식하도록 보장합니다.

### 자막 형식

**기능 설명**：시스템이 인식할 수 있는 자막 파일 형식을 구성하여 시스템이 자막 태그를 올바르게 추가하도록 도와줍니다.

**구성 방법**：자막 파일의 확장자(예: `.srt`, `.ass`, `.idx`)를 입력한 후 **Enter** 키를 눌러 입력을 확인하세요.


### 최소 비디오 크기

**기능 설명**：비디오 파일의 최소 크기 임계값을 설정하면 스캔 시 해당 크기보다 작은 비디오 파일이 무시됩니다.

**단위**：MB

**기본값**：100MB

**특별한 경우**：`.strm` 등의 파일 유형을 사용하는 경우 최소 비디오 크기를 **0MB**로 설정하는 것이 좋습니다. 그렇지 않으면 이러한 파일을 정상적으로 스캔할 수 없을 수 있습니다.

### 중국어 자막 인식

**기능 설명**：중국어 자막의 인식 키워드를 구성하면 비디오 파일 이름에 이러한 키워드가 포함되어 있을 때 시스템이 자동으로 중국어 자막 태그를 추가합니다.

**예시**：`-C`, `.chs` 등. 키워드는 대소문자를 구분하지 않습니다.

### 모자이크 해제 인식

**기능 설명**：모자이크 해제의 인식 키워드를 구성하면 비디오 파일 이름에 이러한 키워드가 포함되어 있을 때 시스템이 자동으로 모자이크 해제 태그를 추가합니다.

**예시**：`-u`, `.uc` 등. 키워드는 대소문자를 구분하지 않습니다.

### 로컬 nfo 읽기

**기능 설명**：활성화하면 시스템이 자동으로 비디오 파일이 위치한 디렉토리의 동명 `.nfo` 파일을 읽고 그 안의 정보를 분석합니다.

**역할**：분석된 정보는 자동으로 미디어 메타데이터를 생성하여 데이터베이스에 저장하므로 메타데이터를 수동으로 편집하는 작업量을 줄일 수 있습니다.

## 설정 예시

다음은 일반적인 스캔 설정 예시입니다:

| 설정 항목 | 설정 값 |
|--------|--------|
| 포맷 제한 | `.mp4`, `.mkv`, `.avi`, `.mov`, `.strm` |
| 디렉토리 무시 | `sample`, `tmp`, `Extras` |
| 파일 무시 | `sample`, `trailer`, `preview` |
| 분할 마크 | `part`, `cd`, `disc` |
| 자막 형식 | `.srt`, `.ass`, `.idx`, `.sub` |
| 최소 비디오 크기 | 100MB |
| 중국어 자막 인식 | `-C`, `.chs`, `.zh` |
| 모자이크 해제 인식 | `-u`, `.uc`, `uncensored` |
| 로컬 nfo 읽기 | 활성화 |

## 자주 묻는 질문

### Q: 일부 비디오 파일이 스캔되지 않는 이유는 무엇입니까?

**A**：가능한 원인은 다음과 같습니다:
- 파일 형식이 포맷 제한 목록에 없음
- 파일이 위치한 디렉토리가 디렉토리 무시 목록에 추가됨
- 파일 이름에 무시된 문자열이 포함됨
- 파일 크기가 설정된 최소 비디오 크기 임계값보다 작음

### Q: 사용자 정의 파일 형식을 올바르게 추가하려면 어떻게 해야 합니까?

**A**：사용자 정의 파일 형식을 추가할 때 다음 사항에 주의하세요:
- 파일 확장자의 점을 반드시 포함해야 합니다(예: `.strm` 대신 `strm`이 아님)
- 각 파일 형식을 개별적으로 추가하고 추가 후 Enter 키를 눌러 확인하세요
- 추가하는 파일 형식이 시스템에서 처리할 수 있는 유형인지 확인하세요

### Q: 분할 마크의 역할은 무엇입니까?

**A**：분할 마크는 시스템이 동일한 비디오 자원에 속하는 여러 분할 파일을 인식하여 이 파일들을 개별 파일로 처리하는 대신 하나의 통합체로 올바르게 처리하도록 돕는 데 사용됩니다.
