# 번호 완성 설정

번호 완성 설정은 번호 식별의 보조 기능으로, 정리 시 식별 결과에 대해 번호 완성을 수행하여 번호 식별의 정확도와 완전성을 향상시킵니다.

<!-- truncate -->

## 번호 완성 설정에 액세스

다음 경로를 통해 번호 완성 규칙 설정 페이지에 액세스할 수 있습니다:

**작업 관리 >> 번호 완성 규칙**

## 설정 인터페이스

![번호 완성 규칙](/img/usage/module/number-complete-01.png)

## 기능 설명

### 번호 완성 규칙

**기능 설명**：번호 완성 규칙을 생성함으로써 시스템은 번호를 식별할 때 자동으로 누락된 부분을 완성하여 번호 식별의 정확도를 향상시킬 수 있습니다.

**적용 시나리오**：
- 파일 이름의 번호가 불완전한 경우
- 번호 형식이 표준화되지 않은 경우
- 번호가 약어 또는 변형을 사용하는 경우

## 설정 방법

### 번호 완성 규칙 생성

1. **번호 완성 규칙** 페이지에서 **생성** 버튼을 클릭합니다
2. 규칙 이름과 완성 규칙을 입력합니다
3. 규칙을 저장합니다

### 규칙 형식

번호 완성 규칙은 일반적으로 두 부분으로 구성됩니다:
- **매칭 패턴**：완성이 필요한 번호를 식별하는 데 사용됩니다
- **완성 패턴**：번호를 어떻게 완성할지 지정하는 데 사용됩니다

## 설정 예

다음은 번호 완성 규칙의 몇 가지 예입니다:

| 규칙 이름 | 매칭 패턴 | 완성 패턴 | 설명 |
|---------|----------|----------|------|
| 접두사 완성 | `^([0-9]+)$` | `ABC-$1` | 순수 숫자 번호를 ABC-숫자 형식으로 완성합니다 |
| 접미사 완성 | `^(XYZ)([0-9]+)$` | `$1-$2` | XYZ123 형식의 번호를 XYZ-123 형식으로 완성합니다 |
| 구분자 대체 | `^([A-Z]+)_([0-9]+)$` | `$1-$2` | 밑줄로 구분된 번호를 하이픈으로 구분된 형식으로 대체합니다 |

## 자주 묻는 질문

### Q: 번호 완성이란 무엇인가요?

**A**：번호 완성은 번호 식별 과정에서 불완전하거나 표준화되지 않은 번호를 자동으로 완성하거나 수정하는 보조 기능으로, 번호 식별의 정확도를 향상시킵니다.

### Q: 언제 번호 완성 규칙을 사용해야 하나요?

**A**：비디오 파일 이름의 번호 형식이 통일되지 않거나 불완전하거나 특별한 형식을 사용하는 경우, 번호 완성 규칙을 사용하면 시스템이 번호를 보다 정확하게 식별할 수 있습니다.

### Q: 번호 완성 규칙이 효과적인지 어떻게判断하나요?

**A**：다음 방법으로 확인할 수 있습니다:
1. 번호 완성 규칙을 설정합니다
2. 스캔 작업을 실행합니다
3. 스캔 결과의 번호 식별 상황을 확인합니다
4. 완성 규칙에 따라 번호가 수정되었는지 확인합니다

### Q: 여러 개의 번호 완성 규칙을 생성할 수 있나요?

**A**：예, 여러 개의 다른 번호 완성 규칙을 생성할 수 있으며, 시스템은 규칙의 우선순위에 따라 적용합니다.

### Q: 번호 완성 규칙의 매칭 순서는 어떻게 되나요?

**A**：시스템은 생성된 순서대로 규칙을 적용하며, 먼저 생성된 규칙이 먼저 적용됩니다.

## 모범 사례

- **규칙 이름**：후속 관리용으로 설명적인 이름을 사용합니다
- **매칭 패턴**：정규 표현식을 사용할 때 패턴이 대상 번호 형식과 정확히 일치하는지 확인합니다
- **완성 패턴**：완성 후 번호 형식이 표준 형식에 준수하는지 확인합니다
- **규칙 수**：규칙 간 충돌을 방지하기 위해 너무 많은 규칙을 생성하지 마세요
- **정기 유지 관리**：실제 사용 상황에 따라 규칙을 정기적으로 조정 및 최적화합니다
