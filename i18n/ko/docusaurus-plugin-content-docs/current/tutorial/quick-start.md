# 빠른 시작

:::info
AMMDS에 오신 것을 환영합니다! 이 가이드는 Windows, 비우 Nas 또는 Docker와 같은 지원되는 방법 중 하나를 사용하여 AMMDS를 배포하는 데 도움이 됩니다.
:::

## 🚀 배포 방법

### Windows 배포

:::note
**패키지 유형**: EXE 설치 프로그램
:::

Windows 사용자를 위해 간단하고 직관적인 실행 가능한 설치 프로그램을 제공합니다:

1. [AMMDS Launcher 릴리스 페이지](https://github.com/QYG2297248353/AMMDS-Launcher/releases)에서 최신 `ammds-setup.exe`를 다운로드합니다
2. 다운로드한 `ammds-setup.exe` 파일을 찾습니다
3. 마우스 오른쪽 버튼을 클릭하고 **관리자로 실행**을 선택합니다
4. 직관적인 화면 마법사를 따라 설치를 완료합니다
5. 설치가 완료되면 서비스가 자동으로 시작됩니다

---

### 비우 Nas 배포

:::note
**패키지 유형**: FPK (비우 패키지)
:::

비우 Nas 사용자를 위해 비우 앱 스토어에 직접 가져오기 위해 특별히 설계된 FPK 패키지를 제공합니다:

1. [비우 Nas 릴리스 페이지](https://github.com/QYG2297248353/AMMDS-Offline-FnNas/releases)에서 최신 `AMMDS.offline.fpk`를 다운로드합니다
2. 비우 Nas 대시보드에 액세스합니다
3. **비우 앱 스토어**를 엽니다
4. **로컬 패키지 가져오기**를 선택하고 다운로드한 FPK 파일을 찾습니다
5. 지침에 따라 설치를 완료합니다
6. 서비스는 비우 Nas 앱 스토어 인터페이스를 통해 관리됩니다

---

### Docker 배포

:::note
**패키지 유형**: Docker 이미지
:::

Docker가 설치된 사용자를 위해 단일 명령으로 AMMDS를 배포합니다:

```bash showLineNumbers title="Docker Command"
docker run -itd \
   --name AMMDS \
   -p 8080:80 \
   -v $(pwd)/data:/ammds/data \
   -v $(pwd)/db:/ammds/db \
   --restart always \
   qyg2297248353/ammds:latest
```

#### Docker 명령 설명

| 옵션 | 설명 |
|--------|-------------|
| `-itd` | 컨테이너를 대화식 모드로 실행하고 가상 TTY를 사용하며 터미널에서 분리합니다 |
| `--name AMMDS` | 컨테이너에 "AMMDS"라는 이름을 할당하여 관리를 용이하게 합니다 |
| `-p 8080:80` | 호스트 포트 8080을 컨테이너 포트 80(웹 인터페이스)에 매핑합니다 |
| `-v $(pwd)/data:/ammds/data` | 애플리케이션 데이터의 영구 스토리지에 사용합니다 |
| `-v $(pwd)/db:/ammds/db` | 데이터베이스 파일의 영구 스토리지에 사용합니다 |
| `--restart always` | 컨테이너가 예기치 않게 중지된 경우 컨테이너를 자동으로 다시 시작합니다 |
| `qyg2297248353/ammds:latest` | 최신 AMMDS Docker 이미지를 사용합니다 |

## 🌐 서비스에 접근

배포에 성공한 후 웹 브라우저를 사용하여 AMMDS에 접근합니다:

1. 좋아하는 웹 브라우저를 엽니다
2. `http://localhost:8080`으로 이동합니다(원격 배포의 경우 `localhost`를 서버 IP로 바꿉니다)
3. AMMDS 로그인 페이지가 표시됩니다

### 📝 기본 자격 증명

| 자격 증명 | 값 |
|------------|-------|
| **사용자 이름** | `ammds` |
| **비밀번호** | `ammds` |

### 💡 첫 접근 팁

:::warning
즉시 로그인 페이지가 표시되지 않으면 기다려 주십시오. 시스템이 아직 백그라운드에서 리소스를 다운로드하고 초기화하고 있을 수 있습니다. 네트워크 속도에 따라 이 과정에 몇 분이 걸릴 수 있습니다.
:::

- 로그인 후 **설정 메뉴에서 즉시 기본 비밀번호를 변경하십시오**
- 직관적인 대시보드를 탐색하여 AMMDS의 기능에 익숙해지십시오
- 오른쪽 아래 모서리의 시스템 상태를 확인하여 모든 서비스가 실행 중인지 확인하십시오
