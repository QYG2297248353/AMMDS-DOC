# NexusPHP

NexusPHP는 절강 대학교의 Nexus 팀에 의해 시작되고 개발된 것으로, 완전하고 체계적이며 사용자 신뢰도와 지식을重視하는 자원 공유 커뮤니티 솔루션을 제공하는 PT(Private Tracker, 즉 프라이빗 Tracker 서버로 BT 다운로드의一种) 오픈 소스 소프트웨어입니다.

:::info 프라이빗 사이트 설명
이 사이트는 초대를 통해 등록해야 하며, 일정한 기준이 있습니다.
:::

## 플러그인 설정

![플러그인 설정](/img/plugin/nexusphp-01.png)

### 기본 설정

- **활성화 상태**：NexusPHP 플러그인 활성화 여부를 제어합니다.
- **검색 기능**：NexusPHP의 검색 기능 활성화 여부，기본값은 비활성화 상태입니다.
- **사용자 정보 동기화**：NexusPHP의 사용자 정보 동기화 기능 활성화 여부，기본값은 비활성화 상태입니다. 사용자 정보는 사이트 랭킹 표시에 사용됩니다.
- **경보 알림**：NexusPHP의 경보 알림 기능 활성화 여부，기본값은 비활성화 상태입니다.

![경보 알림](/img/plugin/nexusphp-02.png)

## NexusPHP 사이트 모듈

오른쪽 상단의「사이트 추가」버튼을 클릭하여 사이트를 추가합니다.

:::warning 버전 요구 사항
NexusPHP 사이트 버전은 1.9.0 이상이어야 합니다.

**버전 확인 방법**：NexusPHP 사이트 페이지 하단 `Powered by NexusPHP`에서 NexusPHP를 클릭하여 사이트 버전을 확인합니다.
:::

![사이트 추가](/img/plugin/nexusphp-03.png)

### 사이트 설정 항목

- **활성화 여부**：이 사이트 활성화 여부，기본값은 비활성화 상태입니다.
- **사이트 이름**：플러그인에 표시되는 사이트 이름입니다. 수동 입력 외에도 드롭다운 메뉴에서 일반적인 사이트 이름을 선택할 수 있으며，시스템이 자동으로 사이트 주소를 채웁니다.
- **사이트 주소**：사이트에 액세스하는 데 사용되는 사이트 주소，기본값은 비어 있습니다.

![사이트 토큰](/img/plugin/nexusphp-04.png)

- **사이트 토큰**：「제어 패널 >> 설정 홈 >> 액세스 토큰」에서 생성 버튼을 통해 생성합니다.

:::info 권한 요구 사항
**기본 권한**：토렌트 목록 가져오기, 토렌트 세부 정보 보기, 토렌트 게시, 사용자 기본 정보 보기

**최소 권한 요구 사항**：토렌트 목록 가져오기, 토렌트 세부 정보 보기, 사용자 기본 정보 보기

**권한不足**：권한이不足한 경우，시스템이 `권한不足`을提示하며，이 사이트를正常하게添加할 수 없습니다.
:::

- **사이트 로고**：플러그인에 표시되는 사이트 로고，기본값은 비어 있습니다.
