# M-Team

M-Team은 다양한 토론 게시판과 결합 그룹을 보유한 종합 토론 포럼으로, 전세계 시사 및 정보 토론을 제공합니다.

프라이빗 사이트: 이 사이트는 초대를 통해 등록해야 하며, 일정한 기준이 있습니다.

<!-- truncate -->

## 플러그인 설정

![플러그인 설정](/img/plugin/m-team-01.png)

활성화 상태: M-Team 플러그인 활성화 여부를 제어합니다.

서비스 주소: M-Team API 서버 주소입니다. 기본값은 `https://api.m-team.io/api`이며, 실제 상황에 따라 수정할 수 있습니다.

현재 M-Team API 서버 주소는 다음 세 개입니다:
- https://api.m-team.cc/api
- https://api.m-team.io/api
- https://api2.m-team.cc/api

키 (Passkey): 인증에 사용되는 키로, "콘솔 >> 설정 >> 홈 >> 키 >> 이메일로 보내기" 버튼을 통해 얻을 수 있으며, 이메일에서 결과를 확인하세요. 기본값은 비어 있습니다.

액세스 토큰 (Token): API 액세스에 사용되는 토큰으로, "콘솔 >> 설정 >> 실험실 >> 액세스 토큰 >> 액세스 토큰 생성" 버튼을 통해 생성할 수 있습니다. 기본값은 비어 있습니다.

![액세스 토큰](/img/plugin/m-team-02.png)

계정 보호 알림: 계정이 14일 이상 다시 로그인하지 않은 경우, 시스템이 알림을 보내 계정을 삭제로부터 보호하기 위해 다시 로그인해야 합니다.
