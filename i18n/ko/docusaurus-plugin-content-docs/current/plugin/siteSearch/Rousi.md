# Rousi

Rousi (肉丝)은 고품질, 고효율의 종합 사이트 검색 서비스를 제공하는 플랫폼입니다.

프라이빗 사이트: 이 사이트는 초대를 통해 등록해야 하며, 일정한 기준이 있습니다.

<!-- truncate -->

## 플러그인 설정

![Rousi 플러그인 설정](./img/plugin/rousi-01.png)

활성화 상태: 플러그인 활성화 여부를 제어합니다.

RSS 서비스 주소: 사이트 도메인 이름, 기본값은 `https://rousi.pro`입니다.

API 서비스 주소: 사이트 API 도메인 이름, 기본값은 `https://rousi.pro/api`입니다.

![Rousi 플러그인 설정](./img/plugin/rousi-02.png)

PassKey: 사이트 API 키로, 인증 및 권한 부여에 사용됩니다. 오른쪽 상단 아바타 >> 계정 설정 -> Passkey에서 얻을 수 있습니다.

계정 보호 알림: 계정 보호 알림을 활성화할지 여부, 기본값은 `false`입니다.
