# R18Dev

R18Dev 是一个源于 R18.com 关闭后的数据保存项目，旨在保留和提供 JAV 影片的元数据信息。通过 R18Dev 插件，AMMDS 可以自动获取影片的标题、演员、发行日期、片商、类别等元数据，帮助你快速构建完整的媒体库信息。

## 插件配置

![基本配置](/img/plugin/r18dev-01.png)

### 基本配置

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 启用状态 | 控制是否启用 R18Dev 插件 | 关闭 |
| 服务地址 | R18Dev 服务的访问地址 | `https://r18.dev` |
| 影视检索 | 控制是否开启影视元数据检索功能 | 关闭 |
| 在线检索 | 开启后优先通过网络检索，而非使用本地导入的数据 | 关闭 |
