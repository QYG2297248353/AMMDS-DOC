# 소개

AMMDS (Adult Movie MetaData Scraper)는 개인 영화 라이브러리를 쉽게 관리할 수 있는 개인 데이터 플랫폼입니다.

## 소개

저희는 기술과 영화를 사랑하는 사람들입니다. 팀원들은 풍부한 소프트웨어 개발 경험을 가지고 있을 뿐만 아니라 영화와 TV 프로그램에 대한 열정도 가지고 있습니다. 그래서 AMMDS라는 플랫폼을 만들게 되었습니다. 개인 미디어 라이브러리를 관리하고 즐기기 위한 전용 플랫폼이죠.

## 프로젝트의 시작

한동안 찾아보니, 시중에 영화 정보를 자동으로 정리하고, 표지와 설명을 채워주며, 여러 기기에서 접근할 수 있는 좋은 도구가 부족하다는 것을 알게 되었습니다. 그래서 직접 AMMDS를 만들기로 했습니다. 컴퓨터에 있는 비디오 파일과 온라인 영화 정보를 자동으로 매칭해 주어, 수동으로 정리하는 번거로움을 덜어줍니다.

## 기능 소개

### 🎬 영화 라이브러리 관리
영화 라이브러리를 쉽게 관리하세요. 다양한 비디오 형식을 지원하며, 자동으로 영화 정보를 가져와 나만의 영화 라이브러리를 구축할 수 있습니다.

### 👤 자동 배우 인식
영화 속 배우를 자동으로 인식하고, 상세한 배우 정보와 작품 목록을 제공하여 쉽게 찾아보고 탐색할 수 있습니다.

### 📄 자동 메타데이터 수집
인터넷에서 영화와 TV 프로그램의 표지, 설명, 평점等信息을 자동으로 가져와 영화 라이브러리를 더욱 풍성하게 만듭니다.

### 📤 공유 기능
친구나 가족과 영화 라이브러리를 공유하여 홈 시어터 경험을 만들 수 있습니다.

### 🔌 플러그인 통합
다양한 플러그인을 설치하여 기존 미디어 서버나 플레이어에 쉽게 연결하고 더 많은 기능을 확장할 수 있습니다.

### 🌐 다중 소스 수집
TMDB, Douban 등 여러 데이터 소스에서 영화 정보를 가져와 데이터의 완전성과 정확성을 보장합니다.

### 🤖 스마트 매칭
스마트 알고리즘을 통해 로컬 파일과 온라인 영화 정보를 자동으로 매칭하여 수동 작업을 줄이고 시간을 절약합니다.

### ⚡ 일괄 처리
여러 비디오 파일을 한 번에 처리하여 관리 효율성을 높입니다.

### 🏷️ 사용자 정의 태그
맞춤형 태그를 지원하여 영화 리소스를 개인별로 분류하고 관리할 수 있습니다.

### 🖥️ 크로스 플랫폼 지원
Windows, macOS, Linux 등 여러 플랫폼을 지원하여 언제 어디서나 영화 라이브러리를 관리할 수 있습니다.
