# 스마트 번역

스마트 번역 플러그인은 AMMDS 플랫폼에 전문적인 다국어 번역 기능을 제공하여 사용자가 다른 언어 간에 정확한 번역을 수행할 수 있도록 지원합니다. 이 플러그인은 고급 기계 번역과 AI 번역 기술을 통합하여 사용자에게 고품질의 문맥 정확한 번역 서비스를 제공하여 다양한 번역 요구를 충족시킵니다.

이 플러그인은 시스템 수준의 번역 솔루션을 채택하여 여러 번역 엔진과 AI 모델을 지원합니다. 사용자는 구체적인 시나리오에 따라 가장 적합한 번역 방법을 선택하여 번역 품질과 정확성을 보장할 수 있습니다.

<!-- truncate -->

## 플러그인 구성

![스마트 번역 플러그인 구성](/img/plugin/translation-01.png)

### 기본 구성 항목

- **번역 활성화**: 스마트 번역 기능을 활성화할지 여부를 제어합니다. 활성화하면 시스템이 구성된 번역 서비스에 따라 번역 요청을 처리합니다.
- **AI 우선**: AI 번역 서비스를 우선적으로 사용할지 여부를 설정합니다. AI 번역을 사용할 수 없는 경우 시스템은 자동으로 기계 번역으로 전환하여 번역 서비스의 연속성을 보장합니다.
- **기본 번역 방법**: 시스템은 기본적으로 MetaTube 번역을 사용합니다. 사용자는 필요에 따라 스마트 번역 기능을 활성화할지 선택할 수 있습니다.

### 기계 번역 추가

![기계 번역 추가](/img/plugin/translation-02.png)

사용자는 여러 기계 번역 서비스를 구성하여 번역 서비스 클러스터를 구축하고 번역의 신뢰성과 다양성을 높일 수 있습니다:

- **공급자 이름**: 번역 엔진에 식별 이름을 설정하여 사용자가 다른 번역 서비스를 관리하고 식별하기 쉽게 합니다.
- **번역 공급자**: Google Translate, 바이두 번역, 텐센트 번역 등과 같은 다른 번역 서비스 공급자를 선택합니다.
- **우선순위**: 번역 엔진의 우선순위 순서를 설정합니다. 시스템은 우선순위에 따라 번역 서비스를 사용합니다. 높은 우선순위 서비스를 사용할 수 없는 경우 자동으로 낮은 우선순위 서비스로 전환됩니다.

> 참고: 다른 번역 공급자를 선택하면 시스템이 해당 구성 항목을 표시합니다. 사용자는 실제 요구에 따라 API 키와 같은 필수 정보를 입력해야 합니다.

### AI 번역 추가

![AI 번역 추가](/img/plugin/translation-03.png)

사용자는 AI 번역 서비스를 구성하여 고급 대규모 언어 모델을 활용하여 더 자연스럽고 정확한 번역 결과를 제공할 수 있습니다:

- **공급자 이름**: AI 번역 서비스에 식별 이름을 설정하여 관리 및 식별을 용이하게 합니다.
- **서비스 유형**: OpenAI 엔드포인트와 Gemini 엔드포인트 두 가지 서비스 유형을 지원하며, 사용자는 실제 상황에 따라 선택할 수 있습니다.
- **서비스 주소**: AI 번역 서비스의 API 주소를 입력하며, 공식 API 주소 또는 자체 구축 프록시 주소를 지원합니다.
- **API 키**: AI 번역 서비스에 액세스하는 키를 입력하여 서비스 호출 보안을 보장합니다.
- **모델**: 사용할 AI 모델을 선택합니다. 다른 모델은 번역 품질과 속도에 차이가 있을 수 있습니다.
- **온도**: AI 번역의 온도 매개변수(0-1 사이)를 설정하여 번역 결과의 창의성을 조정합니다. 온도가 낮을수록 번역 결과가 더 보수적이고 정확하며, 온도가 높을수록 번역 결과가 더 유연하고 다양합니다.

### AI 번역 프롬프트 구성

![AI 번역 프롬프트 구성](/img/plugin/translation-04.png)

사용자는 구체적인 번역 시나리오에 따라 AI 번역 프롬프트를 구성하여 번역 결과를 최적화하고 다른 분야의 전문 요구에 맞게 조정할 수 있습니다:

- **프롬프트**: AI 번역을 안내하는 프롬프트 텍스트를 설정하며 번역 요구에 따라 사용자 정의할 수 있습니다. 예를 들어:
  - 전문 분야 번역: "다음 텍스트를 전문적인 기술 문서 스타일로 번역하고 용어의 정확성을 유지하십시오."
  - 문학 작품 번역: "다음 텍스트를 유창한 문학 스타일로 번역하고 원문의 감정과 정취를 유지하십시오."
  - 비즈니스 문서 번역: "다음 텍스트를 공식적인 비즈니스 스타일로 번역하고 전문 용어의 올바른 사용을 보장하십시오."

### 번역 테스트

![번역 테스트](/img/plugin/translation-05.png)

구성을 완료한 후 사용자는 번역 테스트 인터페이스에서 텍스트를 입력하여 번역 결과를 실시간으로 확인하고 번역 효과를 검증할 수 있습니다. 테스트 기능은 다음을 지원합니다:
- 다국어 상호 번역 테스트
- 다른 번역 엔진 비교 테스트
- 번역 속도 및 품질 평가

## 지원되는 언어

스마트 번역 플러그인은 다음을 포함하여 여러 일반적으로 사용되는 언어의 상호 번역을 지원합니다:
- 중국어(간체/번체)
- 영어
- 일본어
- 한국어
- 프랑스어
- 독일어
- 스페인어
- 러시아어
- 아랍어

구체적으로 지원되는 언어 범위는 구성된 번역 서비스 공급자에 따라 달라집니다.

## 사용 시나리오

스마트 번역 플러그인은 다음 시나리오에 적합합니다:

1. **다국어 콘텐츠 관리**: AMMDS 플랫폼의 다국어 콘텐츠에 번역 지원을 제공하여 다른 언어 버전 간의 콘텐츠 일관성을 보장합니다.
2. **국제화 애플리케이션**: 개발자가 애플리케이션 인터페이스 및 문서를 여러 언어로 번역하여 글로벌 사용자 경험을 향상시키도록 돕습니다.
3. **언어 간 커뮤니케이션**: 다국어 환경에서 커뮤니케이션 콘텐츠를 실시간으로 번역하여 언어 장벽을 제거합니다.
4. **전문 문서 번역**: 기술 문서, 비즈니스 문서 등에 전문적인 번역 서비스를 제공하여 용어 정확성을 보장합니다.
5. **콘텐츠 현지화**: 콘텐츠를 특정 언어 및 문화적 배경에 맞게 조정하여 대상 사용자의 수용도를 높입니다.

## 모범 사례

1. **번역 서비스 혼합 사용**: 번역 콘텐츠의 유형과 중요성에 따라 적절한 번역 서비스를 선택합니다. 전문 용어가 많은 콘텐츠의 경우 AI 번역을 사용하는 것이 좋습니다. 일상 콘텐츠의 경우 속도를 높이기 위해 기계 번역을 사용할 수 있습니다.
2. **번역 서비스 우선순위 합리적 구성**: 신뢰성이 높은 번역 서비스를 높은 우선순위로 설정하여 번역 서비스의 안정성을 보장합니다.
3. **AI 번역 프롬프트 최적화**: 다른 번역 시나리오에 따라 적절한 프롬프트를 사용자 정의하여 번역 품질을 높입니다.
4. **정기적인 번역 서비스 테스트**: 구성된 번역 서비스를 정기적으로 테스트하여 서비스 가용성과 번역 품질을 보장합니다.
5. **번역 사용 현황 모니터링**: 번역 서비스의 사용 현황을 모니터링하여 문제를及时发现하고 해결하며 번역 구성을 최적화합니다.

## 문제 해결

1. **번역 서비스를 사용할 수 없음**: 번역 서비스의 API 키가 올바른지, 네트워크 연결이 정상인지, 서비스 공급자에 서비스 중단이 있는지 확인합니다.
2. **번역 품질 저하**: AI 번역의 온도 매개변수를 조정하고, 프롬프트를 최적화하거나 다른 번역 서비스를 사용해 봅니다.
3. **번역 속도 느림**: 네트워크 연결을 확인하고, 단일 번역의 텍스트 길이를 줄이거나 응답 속도가 더 빠른 번역 서비스를 선택합니다.
4. **번역 결과 불일치**: 동일한 번역 서비스와 구성을 사용하고 있는지 확인하고 다른 시나리오에서 다른 번역 매개변수를 사용하지 않도록 합니다.
