# Rousi

Rousi (肉丝) is a platform specializing in high-quality, efficient comprehensive site search services.

:::info Private Site Note
This site requires registration by invitation and has certain thresholds.
:::

## Plugin Configuration

![Rousi Plugin Configuration](/img/plugin/rousi-01.png)

### Basic Configuration

- **Enable Status**: Controls whether to enable the plugin.
- **RSS Service Address**: Site domain, default is `https://rousi.pro`.
- **API Service Address**: Site API domain, default is `https://rousi.pro/api`.

![Rousi Plugin Configuration](/img/plugin/rousi-02.png)

- **PassKey**: Site API key for authentication and authorization. Get it from the top-right avatar >> Account Settings -> Passkey.
- **Account Protection Notification**: Whether to enable account protection notifications, default is `false`.
