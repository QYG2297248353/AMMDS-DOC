# 高德地图

高德地图插件为 AMMDS 平台首页仪表盘提供天气信息展示。它通过调用高德地图 Web 服务 API 获取实时天气数据，支持两种城市识别方式：手动填写城市代码（adcode）或通过 IP 地址自动定位所在城市。配置简单，即开即用，让你在管理媒体服务的同时随时了解天气状况。
