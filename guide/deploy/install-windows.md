# Windows 安装

::: tip 提示
目前仅在 Windows 10/11 上测试通过，如果在其他版本系统上运行，请自行测试。

自 `v1.6.32` 起，开始支持 Windows 部署。
:::

## 下载安装包

- 访问 [GitHub Releases](https://github.com/QYG2297248353/AMMDS-Docker/releases) 页面，下载最新版安装包。

请注意 `ammds-setup.exe` 文件为安装包文件。

## 安装

需要注意，首次启动时，AMMDS 会自动下载资源数据，根据您的网络情况长短不等，请耐心等待。

只要托盘中程序存在，就说明程序还在下载，并没有出现异常。

如果您的等待时间过长，请检查网络是否正常。或者退出程序后重新启动，会自动清理下载失败的文件，重新开始下载。

下载成功后，后续启动会在几秒内完成，不会再次进行下载。
