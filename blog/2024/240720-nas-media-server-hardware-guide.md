# NAS 媒体服务器硬件选购指南

搭建一台称职的媒体服务器，硬件选择是第一步，也是最关键的一步。本文将从 NAS 系统选型、硬件配置、存储方案等角度出发，为你提供一份实用的选购指南。

## 主流 NAS 系统对比

市面上的 NAS 操作系统各有特色，选择哪一套系统直接决定了后续的使用体验和管理成本。

### 群晖 DSM
拥有最成熟的应用生态和最友好的用户界面，套件中心让软件安装如同手机 App 一样简单。DSM 的媒体相关套件丰富，Video Station、Audio Station、Photo Station 一应俱全。但硬件兼容性受限，仅支持群晖自家设备，性价比偏低。

### QNAP QTS
功能上与群晖不相上下，硬件配置通常更具性价比（同等价位配置更高）。QTS 的虚拟化能力更强，支持 Container Station 和 Virtualization Station。但界面复杂度略高，软件生态的系统整合度不如群晖。

### Unraid
以灵活的存储池管理和 Docker 支持著称，是媒体服务器爱好者的热门选择。Unraid 的存储不受 RAID 级别限制，可以混合不同容量的硬盘使用。内置的 Community Applications 商店让 Docker 部署一键完成。但需要付费授权（基础版 $59 起），且原生文件服务功能不如传统 NAS。

### TrueNAS Scale
基于 Debian Linux 的企业级存储系统，采用 ZFS 文件系统提供卓越的数据完整性保障。Scale 版本原生支持 Docker 和 KVM 虚拟机，适合对数据安全有高要求的用户。但 ZFS 对内存需求较高，配置复杂度也超过其他方案。

## 硬件配置建议

### CPU —— 核心选择

对于媒体服务器而言，CPU 的核心指标不是单核性能，而是**硬件转码能力**。

**推荐首选：Intel N100 / N305**
这两款 Alder Lake-N 系列处理器是当前媒体服务器的性价比之王。它们集成了 Intel UHD Graphics（24/32 EU），完整支持 **Intel QuickSync Video** 技术，可以实现以下编码格式的硬件加速：

- **解码**：H.264、H.265/HEVC、VP9、AV1
- **编码**：H.264、H.265/HEVC

N100 的 TDP 仅 6W，N305 为 15W，功耗极低，非常适合 7×24 小时运行的 NAS 环境。

**进阶选择：Intel i3-12100 / i5-12400**
如果你需要同时处理多个转码任务或运行大量 Docker 容器，桌面级处理器提供了更多的性能余量。它们搭载的 UHD 730/770 核显在转码性能上比 N100 系列更强。

### 存储方案

| 方案 | 优势 | 劣势 | 推荐场景 |
|------|------|------|----------|
| **RAID 5/6** | 存储利用率高，读性能好 | 重建耗时长，有风险 | 大容量媒体存储 |
| **ZFS (RAID-Z)** | 数据完整性校验，快照功能 | 内存需求大，配置复杂 | 重要数据存储 |
| **Unraid 混合池** | 灵活扩容，可混用硬盘 | 写性能受限于校验盘 | 媒体库扩容频繁 |
| **JBOD 直通** | 最大存储空间 | 无冗余保护 | 非重要缓存数据 |

### 内存建议

- **入门配置**：8GB（仅运行媒体服务 + 少量 Docker）
- **标准配置**：16GB（运行媒体栈 + 多容器）
- **ZFS 配置**：32GB 起（ZFS 缓存推荐 1GB 每 TB 存储）

### 网络建议

- **最低要求**：千兆以太网（满足 4K 蓝光原盘串流 ~100Mbps）
- **推荐配置**：2.5GbE（未来 3-5 年不落伍）
- **进阶配置**：10GbE（多用户并发 + 高速传输需求）

## 预算分级推荐

### 入门方案（1500-2500 元）
- 设备：Intel N100 迷你主机（如零刻 EQ12，支持安装 Linux/Unraid）
- 存储：1-2 块 4TB 机械硬盘（JBOD）
- 内存：8GB DDR4/DDR5
- 定位：个人使用，满足基本媒体串流需求
- 缺点：扩展性有限，无法安装更多硬盘

### 进阶方案（4000-6000 元）
- 系统：自组 NAS + Unraid
- CPU：Intel i3-12100
- 主板：B660/B760 ITX 主板
- 存储：2 块 8-12TB 硬盘（Unraid 阵列）
- 内存：16GB DDR4
- 定位：家庭使用，支持多个用户并发访问
- 优势：扩展性强，性能充足

### 发烧方案（10000 元以上）
- 系统：自组 NAS + TrueNAS Scale 或企业级成品 NAS
- CPU：Intel i5-13500 或 Xeon E-2400
- 存储：4+ 块 16-20TB 硬盘（ZFS RAID-Z2）
- 内存：32-64GB ECC
- 网络：2.5GbE/10GbE
- 定位：多用户、高并发、数据安全优先

## 写在最后

硬件选择没有绝对的"最佳方案"，只有最适合你需求的配置。如果你刚开始搭建媒体服务器，不妨从一台 N100 迷你主机加一块大容量硬盘开始，随着媒体库的增长逐步升级。记住，**媒体服务器的核心是软件栈**，好的硬件只是让体验更流畅，而正确的工具链才能让你的媒体库真正井井有条。
