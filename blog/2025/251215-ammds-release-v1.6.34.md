# 发布 AMMDS v1.6.34 — 内置 Metatube 服务

AMMDS v1.6.34 带来了平台支持的重大扩展，新增飞牛 FnOS 应用商店离线安装和 Windows 10/11 安装包支持，同时内置了 Metatube 刮削服务。

{/* truncate */}

## 飞牛 FnOS 应用商店支持

本次更新新增了**飞牛 FnOS 应用商店离线安装支持**。FnOS 用户现在可以通过应用商店直接安装 AMMDS，大大简化了部署流程：

- **离线安装包**：提供适用于 FnOS 的离线安装包，无需联网即可完成安装
- **应用商店集成**：AMMDS 已上架 FnOS 应用商店，用户可一键安装和更新
- **自动更新**：未来版本将通过应用商店通道自动推送更新

## Windows 安装包支持

v1.6.34 正式发布了 **Windows 10/11 安装包**，让更多用户能够便捷地在桌面环境上使用 AMMDS：

- 提供图形化安装向导，引导用户完成安装配置
- 支持 Windows 服务模式，可开机自启动
- 集成管理面板快捷入口，方便日常操作

## 站点认证放宽

根据社区反馈，我们对站点认证策略进行了调整——**仅站点搜索功能需要认证**。这意味着：

- 用户无需认证即可使用大部分基础功能
- 只有涉及资源搜索的模块才需要完成站点认证
- 降低了新用户的上手门槛，提升了使用体验

## 内置 Metatube 服务

**Metatube 服务现已内置于 AMMDS 中**，用户可在设置页面自行开启：

- 内置服务免去了单独部署 Metatube 的麻烦
- 用户可自由选择开启或关闭，灵活控制系统资源占用
- 与 AMMDS 的刮削引擎深度集成，提供更流畅的元数据获取体验

## 问题修复

- 修复了 Windows 安装包在某些系统版本上的兼容性问题
- 修复了 FnOS 离线安装包依赖检查不完整的问题
- 修复了 Metatube 内置服务与代理配置冲突的问题
