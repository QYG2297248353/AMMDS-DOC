# 发布 AMMDS v1.0.30 — 插件助手与分段整理

AMMDS v1.0.30 带来了插件助手项目、密钥管理功能以及影视分段整理功能，同时优化了番号识别逻辑。

{/* truncate */}

## 插件助手

本版本新增了**AMMDS 插件助手**项目，这是一个独立的辅助工具，旨在简化插件的开发、测试和部署流程：

- **插件脚手架**：快速生成插件项目模板，开发者无需从零开始搭建
- **本地测试环境**：提供插件本地运行和调试能力，加速开发迭代
- **打包与发布**：一键打包插件并发布到 AMMDS 插件市场
- **依赖管理**：自动检测并处理插件间的依赖关系

## 密钥管理

新增集中式**密钥管理功能**：

- 统一管理所有第三方服务的 API Key、Token 等敏感信息
- 加密存储，确保凭据安全
- 支持密钥的增删改查操作
- 与各功能模块无缝集成，一处配置全局生效

## 影视分段整理

完成了**影视分段整理功能**，解决多文件影视资源的管理痛点：

- 自动识别同属一部作品的多段视频文件（如 CD1、CD2 或 Part1、Part2）
- 将分段文件合并展示为单一条目，保持媒体库整洁
- 支持自定义分段命名规则，适应不同来源的命名习惯

## 番号识别优化

针对番号（ID）识别逻辑进行了重要优化：

- **修复手动修改识别码后无法匹配的问题**：之前用户手动调整识别码后，系统无法正确重新匹配元数据，现已修复
- **优化特别标签处理**：对番号中 `-C`、`-AI` 等特殊后缀标签的处理逻辑进行了优化，确保这些变体能够被正确识别和匹配，减少因标签格式导致的刮削失败

## 问题修复

- 修复了部分场景下分段文件排序混乱的问题
- 修复了密钥更新后缓存未刷新的问题
- 修复了插件助手生成的项目模板中依赖版本不兼容的问题
