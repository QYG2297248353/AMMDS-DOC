# 平台概述

本文介绍 AMMDS 开放 API 的能力范围、使用流程和基础信息，帮助开发者快速了解和接入开放平台。

{/* truncate */}

AMMDS 开放 API 是一套基于 HTTP 协议的 RESTful 接口，允许开发者通过编程方式集成 AMMDS 的影视元数据管理能力。通过开放 API，您可以实现影视数据的导入、检索、翻译等功能。

## 能力范围

开放 API 当前支持以下功能：

- **影视管理**：影视元数据导入（支持文件上传和 JSON 快捷导入）、从数据源导入、影片详情查询、影片总数获取、影片匹配与检索
- **系统管理**：健康检查、服务状态监控
- **翻译服务**：语言识别、文本翻译

## 使用流程

1. **确认系统版本**：确保 AMMDS 版本支持开放 API 功能
2. **开启开放 API**：在系统全局配置中启用 `GLOBAL_OPEN_API`
3. **创建 API Key**：在系统管理后台生成 API Key
4. **开始集成**：在请求头中携带 API Key 调用接口

## 基础信息

| 项目 | 说明 |
|------|------|
| Base URL | `http://{host}:{port}` |
| 字符编码 | UTF-8 |
| 响应格式 | JSON |
