# NexusPHP

本文介绍 NexusPHP 插件的配置方法，用于 PT 站的站点搜索、用户信息同步和告警通知。

{/* truncate */}

NexusPHP 是由浙江大学的 Nexus 团队开发的一款开源软件，专门用来搭建 PT 站（Private Tracker，即私有种子服务器，是 BT 下载的一种）。它注重用户的信誉和知识分享，帮助大家建立有序的资源分享社区。

:::info[私有站点说明]
这个站需要邀请才能注册，有一定门槛。
:::

## 插件配置

![插件配置](/img/plugin/nexusphp-01.png)

### 基本配置

- **启用状态**：控制是否启用 NexusPHP 插件。
- **检索功能**：是否开启 NexusPHP 的搜索功能，默认是关闭的。
- **同步用户信息**：是否开启 NexusPHP 的用户信息同步功能，默认是关闭的。用户信息用来在站点榜里展示。
- **告警通知**：是否开启 NexusPHP 的告警通知功能，默认是关闭的。

![告警通知](/img/plugin/nexusphp-02.png)

## NexusPHP 站点模块

点右上角「添加站点」按钮，添加站点。

:::warning[版本要求]
NexusPHP 站点版本必须在 1.9.0 以上。

**查看版本方法**：在 NexusPHP 站点页面底部，找到 `Powered by NexusPHP`，点 NexusPHP 就能看到版本号。
:::

![添加站点](/img/plugin/nexusphp-03.png)

### 站点配置项

- **是否启用**：是否启用这个站点，默认是关闭的。
- **站点名称**：站点名称，用来在插件里显示。除了手动输入，也可以在下拉列表里选常见站点名，系统会自动填好站点地址。
- **站点地址**：站点地址，用来访问站点，默认是空的。

![站点令牌](/img/plugin/nexusphp-04.png)

- **站点令牌**：在「控制面板 >> 設定首页 >> 访问令牌」里点创建按钮来生成。

:::info[权限要求]
**默认权限**：获取种子列表、查看种子详情、发布种子、查看用户基本信息。

**最低权限要求**：获取种子列表、查看种子详情、查看用户基本信息。

**权限不足**：如果缺少权限，系统会提示 `权限不足`，无法正常添加这个站点。
:::

- **站点 Logo**：站点图标，用来在插件里显示，默认是空的。
