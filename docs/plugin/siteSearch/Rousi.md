# Rousi

本文介绍 Rousi（肉丝）插件的配置方法，用于高质量综合站点搜索服务。

{/* truncate */}

Rousi（肉丝）是一个专门提供高质量、高效率综合站点搜索服务的平台。

:::info[私有站点说明]
这个站需要邀请才能注册，有一定门槛。
:::

## 插件配置

![Rousi 插件配置](/img/plugin/rousi-01.png)

### 基本配置

- **启用状态**：控制是否启用该插件。
- **RSS 服务地址**：站点域名，默认是 `https://rousi.pro`。
- **API 服务地址**：站点 API 域名，默认是 `https://rousi.pro/api`。

![Rousi 插件配置](/img/plugin/rousi-02.png)

- **密钥（PassKey）**：站点 API 密钥，用来验证身份和授权。在右上角头像 >> 账户设置 -> Passkey 里获取。
- **账户保护通知**：是否开启账户保护通知，默认是 `false`。
