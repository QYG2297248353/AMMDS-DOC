# 智能翻译

智能翻译插件让 AMMDS 平台能翻译多种语言，翻得还挺准。它用了机器翻译和 AI 翻译两种技术，不管你要翻什么，都能搞定。

这个插件是系统层面的翻译方案，支持好几种翻译引擎和 AI 模型。你可以根据实际情况，选最合适的翻译方式。

<!-- truncate -->

## 插件配置

![智能翻译插件配置](/img/plugin/translation-01.png)

### 基本配置项

- **启用翻译**：控制要不要打开智能翻译功能。打开后，系统就会按你配置的方式处理翻译。
- **AI优先**：是否优先用 AI 翻译。如果 AI 翻译用不了，系统会自动换成机器翻译，不影响你使用。
- **默认翻译方式**：系统默认用 MetaTube 翻译。你也可以自己选要不要开启智能翻译。

### 添加机器翻译

![添加机器翻译](/img/plugin/translation-02.png)

你可以配好几个机器翻译服务，组成一个翻译服务集群，这样翻译更可靠、更多样：

- **供应商名称**：给翻译引擎起个名字，方便你管理和辨认。
- **翻译供应商**：选翻译服务商，比如 Google Translate、百度翻译、腾讯翻译等。
- **优先级**：给翻译引擎排个顺序。系统会先用优先级高的，如果不好使，自动换到下一个。

> 注意：选不同的翻译供应商后，系统会显示对应的配置项。你需要按要求填好 API 密钥之类的信息。

### 添加AI翻译

![添加AI翻译](/img/plugin/translation-03.png)

你可以配 AI 翻译服务，用大语言模型翻译，结果更自然、更准确：

- **供应商名称**：给 AI 翻译服务起个名字，方便管理。
- **服务类型**：支持 OpenAI 和 Gemini 两种，你根据情况选。
- **服务地址**：填写 AI 翻译服务的 API 地址，可以用官方地址，也可以用自己搭的代理地址。
- **API密钥**：填访问 AI 翻译服务的密钥，保证调用安全。
- **模型**：选要用的 AI 模型。不同模型翻译质量和速度不一样。
- **温度**：设置 AI 翻译的"温度"参数（0~1之间），调翻译结果的创造性。温度越低，翻译越保守准确；温度越高，翻译越灵活多样。

### 配置AI翻译提示词

![配置AI翻译提示词](/img/plugin/translation-04.png)

你可以根据不同的翻译场景，配 AI 翻译的提示词，让翻译结果更贴合你的需要：

- **提示词**：写一段话告诉 AI 怎么翻译。比如：
  - 专业领域翻译："请把下面的文字翻成专业的技术文档风格，术语要准确。"
  - 文学作品翻译："请把下面的文字翻成流畅的文学风格，保持原文的情感和味道。"
  - 商务文档翻译："请把下面的文字翻成正式的商务风格，专业术语要用对。"

### 翻译测试

![翻译测试](/img/plugin/translation-05.png)

配好之后，你可以在测试界面输入文字，实时看翻译结果，验证效果。测试功能支持：
- 多种语言互相翻译测试
- 不同翻译引擎对比测试
- 翻译速度和质量评估

## 支持的语言

智能翻译插件支持好几种常用语言的互译，包括但不限于：
- 中文（简体/繁体）
- 英语
- 日语
- 韩语
- 法语
- 德语
- 西班牙语
- 俄语
- 阿拉伯语

具体支持哪些语言，看你配的翻译服务商。

## 使用场景

智能翻译插件适合下面这些场景：

1. **多语言内容管理**：给 AMMDS 平台的多语言内容提供翻译，保证各语言版本内容一致。
2. **国际化应用**：帮开发者把应用界面和文档翻成多种语言，让全球用户体验更好。
3. **跨语言沟通**：在多语言环境里，实时翻译交流内容，消除语言障碍。
4. **专业文档翻译**：给技术文档、商务文档等提供专业翻译，保证术语准确。
5. **内容本地化**：把内容适配到特定语言和文化背景，让目标用户更容易接受。

## 最佳实践

1. **混合使用翻译服务**：根据内容类型和重要性选合适的翻译服务。专业术语多的内容，建议用 AI 翻译；日常内容用机器翻译，速度快。
2. **合理配置翻译服务优先级**：把可靠性高的翻译服务设成高优先级，保证翻译稳定。
3. **优化AI翻译提示词**：根据不同的翻译场景，定制合适的提示词，提高翻译质量。
4. **定期测试翻译服务**：定期测试配好的翻译服务，确保能用，质量没问题。
5. **监控翻译使用情况**：盯着翻译服务的使用情况，及时发现问题并解决，优化翻译配置。

## 故障排除

1. **翻译服务用不了**：检查翻译服务的 API 密钥对不对，网络通不通，服务商有没有出问题。
2. **翻译质量不好**：调一下 AI 翻译的温度参数，优化提示词，或者换一个翻译服务试试。
3. **翻译速度慢**：检查网络，减少单次翻译的文字量，或者换成响应更快的翻译服务。
4. **翻译结果不一样**：确保用同一个翻译服务和配置，别在不同场景下用不同的翻译参数。
