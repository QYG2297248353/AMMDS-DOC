# 麻豆传媒映画

麻豆传媒映画是一家做成人内容制作和分发的公司，注册在美国加利福尼亚州，服务器也在美国。这个插件让 AMMDS 能采集麻豆传媒的影视和演员信息。

## 插件配置

![插件配置](/img/plugin/madou-01.png)

### 基本配置

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 启用状态 | 控制是否启用麻豆传媒映画插件 | 关闭 |
| 服务地址 | 麻豆传媒的 API 服务地址 | `https://api.modelmediaasia.com` |
| 影视采集 | 控制是否采集麻豆传媒的影视信息（用了提前缓存机制） | 关闭 |
| 演员采集 | 控制是否采集麻豆传媒的演员信息（用了提前缓存机制） | 关闭 |

### 配置说明

- **服务地址**：默认是官方 API 地址，如果想用代理或者自建服务，可以在这里改
- **元数据采集**：用了提前缓存机制，数据获取更快更稳定
- **数据同步**：开启采集后，系统会定期同步最新的影视和演员信息

## 注意事项

:::note 数据采集机制
插件先把元数据缓存好，这样就算网络不太稳定，也能正常使用。
:::

:::info 服务可用性
因为服务器在美国，建议保证网络连接稳定。如果慢，可以配个代理来提速。
:::
