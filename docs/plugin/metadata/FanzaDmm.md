# Fanza DMM

FANZA 是日本最大的成人内容平台，可以在线卖 DVD、租影片、看流媒体。它和很多知名片商合作，有专属频道，还有包月随便看的「見放題 ch」（随心看）服务，平台上大部分视频都能看。

## 插件配置

![插件配置](/img/plugin/fanzadmm-01.png)

### 基本配置

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 启用状态 | 控制是否启用 Fanza DMM 插件 | 关闭 |
| Ranking 榜单数据 | 控制是否采集 FANZA 的排行榜数据 | 关闭 |
| 影视检索 | 控制是否开启 FANZA 的影视内容搜索功能 | 关闭 |

### 配置说明

- **榜单数据采集**：开启后会定期同步 FANZA 各个分类的排行榜，给你的媒体库提供热门内容参考
- **影视检索**：开启后可以通过 FANZA 数据源搜索影视内容，提高元数据匹配的准确度
- **数据更新频率**：系统会根据配置自动调整数据更新周期，保证数据是最新的

## 功能特性

- **多维度数据**：包括影片、演员、排行榜等各种数据
- **实时同步**：支持定期同步平台最新数据
- **精准匹配**：基于 FANZA 官方数据，元数据匹配精度高

## 注意事项

:::info 地域限制
FANZA 服务可能有地域访问限制，部署时要注意网络环境。
:::

:::note 数据来源
插件数据来自 FANZA 官方 API，数据合法、准确。
:::
