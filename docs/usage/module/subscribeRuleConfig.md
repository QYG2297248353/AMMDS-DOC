# 订阅规则配置

本文介绍 AMMDS 的订阅规则配置，包括中字优先、搜索顺序和下载目录等设置项说明。

{/* truncate */}

订阅规则配置用来告诉系统：你想看什么样的资源、从哪找、下到哪，让系统按你的偏好自动干活。

## 访问订阅规则配置

通过以下路径进入：

**任务管理 >> 订阅规则**

## 配置界面

![订阅规则配置](/img/usage/module/subscribe-config-01.png)

## 配置项说明

### 中字优先

**干啥的**：打开后，系统会优先搜带中文字幕或中文配音的版本。

**啥时候用**：如果你更习惯看中文，就打开这个，拿到中文版本的概率更高。

### 搜索顺序

**干啥的**：设置搜索源的优先级。系统会按你排的顺序，依次去各个搜索源找资源。

**建议**：把资源质量高、更新快的搜索源排前面，这样更快找到好资源。

### 下载目录

**干啥的**：指定下载器把文件下到哪个文件夹。

**重要提醒**：

:::warning[注意事项]
- 这个路径跟 AMMDS 系统本身没啥关系，要根据你用的下载器的配置来设置
- 确保这个文件夹存在，并且有写入权限
- 建议用绝对路径（比如 `D:\Downloads\Movies`），别用相对路径，免得路径搞错
:::

## 配置示例

下面是一个典型的订阅规则配置：

1. **中字优先**：开启
2. **搜索顺序**：搜索源 A → 搜索源 B → 搜索源 C
3. **下载目录**：`D:\Downloads\Movies`

## 常见问题

### Q: 开了中字优先，为啥还是下了没中字的版本？

**A**：可能是因为这个资源目前还没出中字版。系统在没有中字版本的情况下，会先下其他版本，确保你能及时拿到资源。

### Q: 下载目录设错了会咋样？

**A**：可能导致：
- 下载器没法正常存文件
- 系统找不到下好的文件
- 硬盘空间不够时下载失败

所以一定要把下载目录路径设对。
