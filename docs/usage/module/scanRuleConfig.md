# 扫描配置

在任务执行前，对扫描规则进行配置。

<!-- truncate -->

通过 任务管理 >> 扫描规则配置 进入扫描规则配置页面。更新配置记得点击 保存 按钮。否则配置不会生效。

![扫描规则配置](/img/usage/module/scan-config-01.png)

格式限制：
    当前内置了一些常见的媒体文件类型，如：.mp4, .mkv, .avi, .mov 等。
    用户可以根据需要，添加其他文件类型。添加时，输入文件类型的扩展名，如：.strm 等后按下 回车 (Enter) 键确认输入。

忽略目录：
    用户可以配置一些目录，扫描时会忽略这些目录。例如：sample, tmp 等。该目录下以及其子目录下的文件，都不会被扫描到。
    忽略目录时，需要输入目录的相对路径。例如：sample, tmp 等。不需要输入完整路径，目录名称忽略大小写。

忽略文件：
    用户可以配置一些文件，扫描时会忽略这些文件。例如：sample hh800.com@ 等。该文件不会被扫描到。
    忽略文件时，需要输入文件名称中包含的字符串。例如：sample, hh800.com@ 等。不需要输入完整文件名，字符串忽略大小写。

分段标记：
    一些分段视频文件，文件名中会包含一些标记，如：part, cd 等。辅助系统进行分段视频标记，避免整理时将其作为单独的文件处理。
    分段标记时，需要输入分段标记的字符串。例如：part, cd 等。不需要输入完整文件名，字符串忽略大小写。

字幕格式：
    用户可以配置一些字幕文件格式，辅助系统正确增加字幕标签。例如：.srt, .ass, .idx 等。
    字幕格式时，需要输入字幕文件的扩展名，如：.srt, .ass, .idx 等后按下 回车 (Enter) 键确认输入。

![扫描规则配置](/img/usage/module/scan-config-02.png)

最小视频大小：
    用户可以配置一个视频文件的最小大小。扫描时，小于该大小的视频文件，会被忽略。单位：MB。默认值为 100MB。
    如果用户通过 .strm 等文件类型的视频，建议将最小视频大小设置为 0MB。否则，可能会导致无法正常扫描到视频文件。

中文字幕识别：
    当视频文件名中包含该关键词时，会自动增加中文字幕标签。例如：-C .chs 等。关键词忽略大小写。

马赛克破解识别：
    当视频文件名中包含该关键词时，会自动增加马赛克破解标签。例如：-u .uc 等。关键词忽略大小写。

读取本地 nfo：
    如果视频文件所在目录下，存在同名的 .nfo 文件，会自动读取并解析该文件。
    解析到的信息，会自动创建媒体元数据，存储到数据库中。
