# 扫描配置

本文介绍 AMMDS 的扫描配置，包括格式限制、忽略目录和文件识别规则等设置项说明。

{/* truncate */}

扫描配置用来告诉系统：扫描文件的时候，什么文件要处理、什么要跳过，怎么识别影视资源。

## 访问扫描配置

通过下面的路径就能找到扫描规则配置页面：

**任务管理 >> 扫描规则配置**

:::info[操作提示]
改完配置记得点 **保存**，不然不会生效。
:::

## 配置项说明

### 格式限制

**干啥的**：告诉系统哪些文件类型才算媒体文件，系统只认这些格式。

**默认已有**：系统自带了一些常见的格式，比如 `.mp4`, `.mkv`, `.avi`, `.mov` 这些。

**自己加**：想加新的就输入扩展名（比如 `.strm`），然后按回车键确认。

### 忽略目录

**干啥的**：有些目录你不想被扫描（比如放样片、临时文件的文件夹），可以在这里设置跳过。

**怎么配**：输入目录的相对路径（比如 `sample`, `tmp`），不区分大小写。

**啥场景用**：比如下载文件夹里有个 `sample` 目录放的只是样片，不想扫它就可以加进来。

### 忽略文件

**干啥的**：有些文件名里带了特定字眼（比如 `sample`），你不想处理这些文件，可以在这里设置。

**怎么配**：输入文件名里包含的字符串（比如 `sample`, `hh800.com@`），不区分大小写。

**啥场景用**：比如有些文件名字里带了广告标记，你想忽略掉。

### 分段标记

**干啥的**：有的视频被拆成了好几段（比如 CD1, CD2），系统通过这个标记知道它们是一起的，不会当成单独的电影处理。

**怎么配**：输入分段标记（比如 `part`, `cd`），不区分大小写。

**啥场景用**：电影分上下集的、或者分成了多个文件的情况。

### 字幕格式

**干啥的**：设置系统能识别哪些字幕文件格式。

**怎么配**：输入字幕文件扩展名（比如 `.srt`, `.ass`, `.idx`），然后按回车键确认。

### 最小视频大小

**干啥的**：设置一个大小门槛，比这个小的视频文件会被忽略（太小的多半不是正经电影）。

**单位**：MB

**默认值**：100MB

**特殊情况**：如果你用的是 `.strm` 这类文件（只是个链接，不占空间），建议把最小值设成 **0MB**，不然扫不到它们。

### 中文字幕识别

**干啥的**：视频文件名里如果包含某些关键词，系统会自动给它打上"中文字幕"的标签。

**示例**：`-C`, `.chs` 等，不区分大小写。

### 马赛克破解识别

**干啥的**：视频文件名里如果包含某些关键词，系统会自动给它打上"破解"标签。

**示例**：`-u`, `.uc` 等，不区分大小写。

### 读取本地 nfo

**干啥的**：打开这个功能后，系统会自动读取视频文件旁边同名的 `.nfo` 文件，把里面的信息解析出来。

**有啥用**：解析到的信息会自动存到数据库里，省得你再手动去填影片信息了。

## 配置示例

下面是一个典型的扫描配置：

| 配置项 | 设置值 |
|--------|--------|
| 格式限制 | `.mp4`, `.mkv`, `.avi`, `.mov`, `.strm` |
| 忽略目录 | `sample`, `tmp`, `Extras` |
| 忽略文件 | `sample`, `trailer`, `preview` |
| 分段标记 | `part`, `cd`, `disc` |
| 字幕格式 | `.srt`, `.ass`, `.idx`, `.sub` |
| 最小视频大小 | 100MB |
| 中文字幕识别 | `-C`, `.chs`, `.zh` |
| 马赛克破解识别 | `-u`, `.uc`, `uncensored` |
| 读取本地 nfo | 开启 |

## 常见问题

### Q: 为啥有些视频文件扫不到？

**A**：可能的原因有：
- 文件格式不在你设置的格式限制列表里
- 文件所在的目录被你加了"忽略目录"
- 文件名里包含被你忽略的字符串
- 文件太小，没达到你设置的最小视频大小

### Q: 怎么正确加自定义文件格式？

**A**：加的时候注意几点：
- 扩展名要带点号（比如 `.strm`，不是 `strm`）
- 每个格式单独加，加完按回车确认
- 确保加的格式系统能处理

### Q: 分段标记有啥用？

**A**：分段标记帮系统识别属于同一部电影的几个分段文件，让它们被当成一个整体处理，而不是当成多个不相干的文件。
