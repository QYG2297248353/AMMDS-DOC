# 执行刮削任务

刮削任务是整理电影的第二步。简单说就是：系统根据刮削规则，去网上找到电影的信息（封面、简介、演员等），然后整理好放到指定位置。

> **"刮削"是啥意思？** 刮削其实就是系统自动去网上搜刮电影的信息，比如封面图、剧情简介、演员表、评分这些（统称"元数据"），然后下载下来跟你的视频文件配对。就像给电影文件"贴标签"一样。

## 前置条件

在执行刮削任务前，先确认下面这些准备好了：

### 网络连接

整理文件需要上网找电影信息，所以要确保网络通畅。

### 元数据插件

确认已经开启并配好了相关的元数据插件，这些插件是帮你找到电影信息的关键工具：

+ [Metatube 插件](/docs/plugin/metadata/Metatube)
+ [ThePornDB 插件](/docs/plugin/metadata/ThePornDB)
+ [StashBox 插件](/docs/plugin/metadata/ThePornDB)
+ [FanzaDMM 插件](/docs/plugin/metadata/FanzaDmm)
+ [Madou 插件](/docs/plugin/metadata/Madou)

:::warning 重要提醒
没开启元数据插件，就拿不到电影信息，刮削必然会失败。
:::

:::info 数据来源声明
本软件不提供任何源数据，所有电影信息都来自你自己配置的插件服务。
:::

## 执行刮削任务

### 第 1 步：进入刮削入库页面

从左侧导航栏进 **任务管理**，然后选 **刮削入库**。

![刮削入库页面](/img/usage/scrape/task-scrape-01.png)

### 第 2 步：选择要刮削的文件

在刮削入库页面，你能看到所有待刮削的文件列表。勾选你要处理的文件。

### 第 3 步：开始刮削

点 **刮削整理** 按钮，系统就开始干活了。

![执行刮削任务](/img/usage/scrape/task-scrape-02.png)

### 第 4 步：查看任务状态

任务执行过程中，页面会实时显示进度：
- 刮削成功的文件数
- 刮削失败的文件数
- 跳过的文件数

### 第 5 步：终止任务（如果需要）

执行过程中如果想停，可以随时点 **取消** 按钮。

## 刮削原理说明

刮削的整个流程是这样的：

1. **读取刮削规则**：系统根据你配的规则，决定怎么处理文件
2. **匹配元数据**：通过你开启的元数据插件，试着找到电影的详细信息
3. **生成影视文件**：根据找到的信息，生成标准化的影视文件
4. **整理文件结构**：按你预设的命名规则和目录结构，把文件整理好

## 常见问题与解决方案

### 问题：刮削失败

**可能原因**：
- 网络连不上
- 元数据插件没开或配错了
- 刮削规则设置不对
- 文件命名不符合识别规则

**解决方法**：
- 检查网络是不是正常
- 确认元数据插件已开启且配置正确
- 检查刮削规则设置是否合理
- 试着改一下文件命名，让它更容易被识别

### 问题：电影信息不全

**可能原因**：
- 插件服务暂时用不了
- 这个电影在插件服务里找不到
- 网络慢导致获取超时

**解决方法**：
- 等会儿再试，可能只是暂时抽风
- 换别的元数据插件试试
- 检查网络稳不稳定

## 最佳实践

1. **分批处理**：文件多的话建议分批刮削，别一次搞太多，免得系统卡死
2. **先配好再干**：刮削之前，确保刮削规则和元数据插件都配好了
3. **定期更新插件**：插件更新了匹配效果更好
4. **看日志查错**：刮削失败时，去系统日志里看看详细的错误信息

## 注意事项

:::caution 注意
刮削任务可能要跑挺久，具体取决于文件数量和网速，耐心等着就行。
:::

:::tip 提示
想让刮削效果更好，建议文件名尽量规范，包含必要的识别信息（比如影片编号、标题等）。
:::
