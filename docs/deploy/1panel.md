# 1Panel 部署指南

1Panel 是一个服务器管理面板，可以通过网页界面来管理应用、网站、文件、数据库以及大语言模型（LLMs）等资源。下面介绍如何在 1Panel 上部署 AMMDS。

<!-- truncate -->

## 安装第三方应用商店

本方案适用于 1Panel v1 和 v2 版本。

### 自动化安装

自动化安装每 3 小时自动更新一次第三方应用商店内容，确保你始终能获取到最新的应用。

#### 安装脚本

```bash
curl -sSL https://install.lifebus.top/auto_install.sh | bash
```

#### 卸载脚本

```bash
curl -sSL https://install.lifebus.top/auto_uninstall.sh | bash
```

### 手动安装

手动安装只会拉取一次最新的应用商店内容，后续更新需要再次执行此脚本。

```bash
curl -sSL https://install.lifebus.top/app_install.sh | bash
```

## 更新商店内容

### 操作步骤

1. **进入商店设置**
   通过左侧导航栏进入应用商店，点击进入商店设置页面。


2. **执行更新操作**

   ![更新商店内容](/img/deploy/install-1panel-00.png)
   
   #### v1 版本
   在商店页面的右上角点击「更新应用列表」按钮。

   ![更新商店内容](/img/deploy/install-1panel-01.png)

   #### v2 版本
   在商店页面的左上角点击「同步本地应用」按钮。

## 安装应用

### 应用搜索与选择

![安装](/img/deploy/install-1panel-02.png)

1. **搜索 AMMDS 应用**
   在应用商店中搜索「ammds」。

2. **版本选择**
   系统会显示两个版本：
   - **兼容版**：适合配置较低的设备
   - **最新版**：适合配置较高的设备

   ![安装](/img/deploy/install-1panel-03.png)

### 安装参数配置

点击安装按钮后，会弹出参数配置面板。

#### 基本配置

- **名称**：建议填「ammds」，也可以自己改
- **版本**：建议选「最新版」
- **网络模式**：没啥特殊需求的话，保持默认就行
- **数据持久化路径**：填服务器上的绝对路径，用来存 AMMDS 的数据
- **WebUI 端口**：建议填 8080，也可以自己改
- **API 端口**：建议填 8081，也可以自己改

其他参数可选，按需填写就行。

![安装](/img/deploy/install-1panel-04.png)

#### 挂载目录设置

**自定义挂载目录**：建议填媒体文件所在的目录，可选项。

![安装](/img/deploy/install-1panel-05.png)

#### 端口访问配置

**必须勾选「端口外部访问」**，否则没法通过网络访问 WebUI 和 API。

点击「确认」，开始安装。

## 安装完成

![安装完成](/img/deploy/install-1panel-06.png)

### 验证安装成功

安装完成后，在「已安装」选项卡里就能看到 AMMDS 了。

### 管理功能

应用卡片上提供了多种快捷操作：
- **终端**：进入应用容器终端
- **日志**：查看运行日志
- **启动/停止/重启**：管理应用服务
- **卸载**：移除应用
- **重建**：重新构建应用容器

## 访问应用

![访问应用](/img/deploy/install-1panel-07.png)

### 访问方式

安装完成后，可以通过浏览器访问 AMMDS 的 WebUI 和 API：

- **WebUI 地址**：http://&lt;服务器IP&gt;:&lt;WebUI端口&gt;
- **API 地址**：http://&lt;服务器IP&gt;:&lt;API端口&gt;

### 默认登录信息

- **用户名**：ammds
- **密码**：ammds

:::info 安全提示
首次登录后，请立即修改默认密码，确保安全。
:::

## 警告和注意事项

### 安全警告

:::danger 重要安全提示

- **官方来源**：只从官方渠道获取安装脚本，别用第三方来源
- **密码安全**：设置强密码，定期更换
- **端口安全**：修改默认端口，降低被攻击的风险
- **外部访问**：按需配置端口外部访问，别不必要地暴露

:::

### 系统要求

:::info 系统配置要求

#### 最低配置

- 操作系统：Linux 发行版（支持 1Panel 安装）
- 处理器：Intel Core i3 或同等性能
- 内存：4GB RAM
- 磁盘空间：至少 10GB 可用空间
- 网络：宽带互联网连接

#### 推荐配置

- 操作系统：Linux 发行版（支持 1Panel 安装）
- 处理器：Intel Core i5 或同等性能
- 内存：8GB RAM 或更多
- 磁盘空间：20GB 可用空间
- 网络：稳定的宽带互联网连接

:::

### 使用注意事项

- **数据备份**：定期备份配置文件和重要数据
- **资源管理**：留意服务器资源使用情况
- **网络连接**：下载资源和更新时，确保网络稳定
- **系统维护**：定期更新 1Panel 和相关应用

### 故障排除

#### 常见问题

- **安装失败**：检查网络连接和磁盘空间
- **启动异常**：查看应用日志获取错误信息
- **访问失败**：检查端口配置和防火墙设置
- **升级失败**：试试先卸载旧版再安装新版

#### 联系方式

遇到解决不了的问题，请到官方 GitHub 仓库提交 Issue。
