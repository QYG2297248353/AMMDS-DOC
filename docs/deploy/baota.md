# 宝塔面板

宝塔面板（bt.cn）就是一个服务器管理工具，把那些复杂的命令操作变成了网页上的按钮和开关，点点鼠标就能管理网站、数据库、SSL 证书什么的，不用再记一堆命令了。

<!-- truncate -->

## 准备工作

在开始之前，请先确认以下两样东西已经准备好：

1. **已经装好了宝塔面板** — 如果还没装，先去 [bt.cn](https://bt.cn) 照着装一个
2. **宝塔里已经装了 Docker** — 打开宝塔面板，在左侧软件商店里搜索 "Docker"，安装即可

## 用宝塔的 Docker 部署 AMMDS

### 第一步：打开 Docker 管理

在宝塔面板左侧菜单找到 **Docker**，点进去。

### 第二步：创建容器

1. 在 Docker 管理页面，点击 **容器** → **创建容器**
2. 按下面的信息填写：

| 设置项 | 填写内容 | 大白话解释 |
|--------|---------|-----------|
| **镜像名称** | `qyg2297248353/ammds:latest` | 这是 AMMDS 的"安装包"，填这个地址会自动下载 |
| **容器名称** | `AMMDS` | 给你的 AMMDS 起个名字，方便认出来 |
| **端口映射** | 主机端口 `8080` → 容器端口 `80` | 把服务器的 8080 端口分给 AMMDS 用，访问时输入 `http://你的IP:8080` |
| **挂载目录** | 见下方说明 | 把服务器上的文件夹"映射"到容器里，这样下载的电影文件不会因为容器重启而丢失 |

### 第三步：设置挂载目录（重要！）

挂载目录就是把你服务器上的文件夹"借"给 AMMDS 用。最少需要挂载以下内容：

- **数据目录**：把服务器的某个文件夹（比如 `./data`）挂载到容器的 `/ammds/data`
- **数据库目录**：把服务器的某个文件夹（比如 `./db`）挂载到容器的 `/ammds/db`
- **下载目录**：把服务器的某个文件夹（比如 `./download`）挂载到容器的 `/ammds/download`
- **媒体目录**：把你存电影电视剧的文件夹（比如 `./media`）挂载到容器的 `/media`

:::note
**注意：** 存放电影的文件夹**不要**挂载到 `/ammds` 开头的路径下，否则数据可能会丢。

**举个例子：** 如果你的电影在服务器的 `/mnt/movie` 文件夹里，那就这样挂载：`/mnt/movie:/mnt/movie`
:::

### 第四步：设置自动重启

在创建容器的页面，找到 **重启策略**，选 **"总是重启（always）"**。这样就算服务器重启了，AMMDS 也会自己重新跑起来。

### 第五步：启动容器

填好上面所有信息后，点击确认创建。宝塔就会自动拉取镜像并启动容器。

### 第六步：访问 AMMDS

打开浏览器，输入：

```
http://你的服务器IP:8080
```

比如你的服务器 IP 是 `192.168.1.100`，那就输入 `http://192.168.1.100:8080`。

### 默认账号密码

- **用户名**：`ammds`
- **密码**：`ammds`

:::tip
如果看不清文字，可以切换到浅色模式。
:::

## 注意事项

- **防火墙**：如果打不开页面，检查一下服务器的防火墙是不是把 8080 端口放行了，宝塔面板的安全页面里可以添加端口放行规则
- **数据持久化**：挂载目录非常重要！不挂载的话，一旦容器删了，你所有的配置和数据就全没了
- **时区设置**：如果需要修改时区，可以在创建容器时添加环境变量 `TZ=Asia/Shanghai`（东八区时间）
