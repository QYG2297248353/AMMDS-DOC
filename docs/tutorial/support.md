# 支持我们

🥳 让我们一起成长！

欢迎来到 AMMDS 支持页面！感谢你的关注和支持。你的每一份支持都会帮我们改进 AMMDS，给大家带来更好的体验。

## 怎么支持我们？

### 1. 成为会员

去我们的在线商店购买 AMMDS 相关产品或服务。既能拿到好产品，又能支持我们的工作。

### 2. 分享给朋友

觉得 AMMDS 好用？向家人朋友推荐一下吧！分享就是最好的鼓励，也能让更多人用上这个工具。

### 3. 贡献数据

AMMDS 的数据是共享的，谁都可以查看、修改和提改进建议。如果遇到问题，可以在 GitHub 上反馈。

### 4. 提建议

你的意见对我们很重要。不管是想要新功能、发现 bug，还是有其他想法，都可以告诉我们。可以通过官网联系，或者直接在 GitHub 上提交问题。

### 5. 赞助我们

如果你喜欢我们的工作，想支持我们继续做下去，可以考虑捐赠。你的支持能帮我们付服务器、域名和其他必要的开销。

**首选赞助平台：**

- 在线商店（支付宝）：[足迹店铺](https://store.lifebus.top/)
